## Comprendre la mécanique des coffres-forts dans Resurgence

Dans *The Division Resurgence*, les coffres-forts servent d'énigmes environnementales. Contrairement aux caisses de butin standard qui nécessitent une simple interaction, ces boîtes verrouillées exigent une combinaison à trois ou quatre chiffres. Le jeu fournit généralement la solution de deux manières :

1.  **Indices environnementaux :** Des codes écrits sur des tableaux noirs, des affiches ou des enseignes au néon à proximité immédiate du coffre.
2.  **Indices cryptiques (Objets de collection) :** Des éléments de renseignement spécialisés trouvés près du coffre qui doivent être lus dans votre menu « Base de données » sous l'onglet « Indices cryptiques ».

> **💡 Conseil :** Si vous trouvez un coffre mais ne voyez aucun code à proximité, vérifiez votre mini-carte pour une icône d'objet de collection. Ramasser ces « Indices cryptiques » révèle souvent la combinaison à travers une énigme ou une date spécifique mentionnée dans le texte.

| Type de coffre | Emplacement de l'indice | Niveau de récompense typique |
| :--- | :--- | :--- |
| **Coffre de colonie** | Tableaux noirs ou murs | Équipement supérieur (violet) |
| **Coffre de planque** | Affiches ou panneaux | Matériaux et ressources de fabrication |
| **Coffre du monde ouvert** | Objets de collection (Indices) | Potentiel de haute qualité (or) |

### Coffre de la colonie Spring Blossom
C'est probablement le premier coffre que vous rencontrerez. Après un voyage rapide vers la colonie de Spring Blossom, dirigez-vous vers les niveaux supérieurs du bâtiment principal. Vous trouverez un petit espace de bureau avec un coffre posé sur le sol.
*   **Le Code :** 3580
*   **L'indice :** Regardez le tableau noir immédiatement à droite du coffre. Les chiffres sont écrits clairement à la craie blanche.

### Dortoirs de la Base d'Opérations (BoO)
À l'intérieur de la base d'opérations principale de New York, dirigez-vous vers la section des dortoirs où les agents se reposent.
*   **Le Code :** 1350
*   **L'indice :** Avant d'entrer dans la pièce avec le coffre, vérifiez le panneau d'affichage dans le couloir. Un petit morceau de papier épinglé au panneau contient la séquence à quatre chiffres.

## Explorer les rues : Coffres du périmètre de la BoO

La zone entourant la Base d'Opérations regorge de butin. Il y a trois coffres spécifiques situés à quelques pas de l'entrée principale.

### Le magasin Oaklock et le supermarché
Sortez de la Base d'Opérations et tournez immédiatement à droite. Suivez la rue jusqu'à ce que vous voyiez le magasin Oaklock sur votre gauche et un supermarché sur votre droite.
1.  **Magasin Oaklock :** Entrez dans le magasin et regardez derrière le comptoir. Le coffre est au sol. Le code **122** se trouve sur une enseigne décorative au mur.
2.  **Supermarché :** Traversez la rue et entrez dans le supermarché. Dirigez-vous vers le mur du fond. Le coffre est dissimulé, et le code **1987** est affiché sur un panneau directement au-dessus de l'unité.

### Le coffre « Anniversaire » de Serotentia
Ce coffre est un peu plus complexe. Depuis l'entrée de la BoO, tournez à gauche et descendez la rue après l'hôpital de campagne Madison. Vous trouverez un grand coffre à l'intérieur d'une tente/bâtiment médical étiqueté Serotentia.
*   **Le Code :** 1105
*   **L'indice :** Vous devez d'abord ramasser un objet de collection sur le sol à l'extérieur de la tente. Cet objet est intitulé « Cadeau d'anniversaire ». En lisant l'entrée dans votre base de données, une date est mentionnée : le 5 novembre. Au format américain (MM/JJ), cela se traduit par 1105.

> **Attention :** Certains joueurs confondent le format de la date. Si 1105 ne fonctionne pas pour votre version régionale spécifique, essayez 0511, bien que 1105 soit la norme pour la sortie mondiale.

## Caches des planques

Les planques ne servent pas seulement à se réapprovisionner en munitions ; elles abritent souvent des récompenses cachées pour les agents observateurs.

### Planque Autumn's Hope
Localisez le PNJ nommé Mullins au bureau de réception.
*   **Le Code :** 1883
*   **L'indice :** Regardez la date « Established » (Établi en) sur le panneau suspendu derrière le bureau. Elle peut être difficile à lire selon vos paramètres graphiques, mais l'année est 1883.

### Planque Dante's Run
Sortez de la salle principale de la planque par la porte latérale près du mannequin. Cela mène à un magasin de vêtements attenant.
*   **Le Code :** 7716
*   **L'indice :** Faites le tour jusqu'à l'avant du comptoir de caisse. Une affiche promotionnelle présente le code en gros caractères.

### Planque de la station Kerman
En sortant de la salle d'apparition principale de la station Kerman, marchez directement vers la zone des casiers étiquetée « Safia Nerdies ».
*   **Le Code :** 7069
*   **L'indice :** Une petite note adhésive jaune est épinglée sur le côté des casiers juste à côté du coffre.

## Conseils pour trouver les futurs codes de coffre-fort

À mesure que le jeu s'étendra en 2026, de nouvelles régions introduiront probablement davantage de boîtes verrouillées. Gardez ces conseils à l'esprit lorsque vous explorez de nouvelles zones :

1.  **Consultez la base de données :** Si vous trouvez un coffre et aucun code évident, ouvrez votre menu et regardez dans « Objets de collection » -> « Indices cryptiques ». Si vous avez ramassé des renseignements récemment, la réponse s'y trouve probablement.
2.  **Recherchez les contrastes élevés :** Les développeurs placent souvent les codes dans des couleurs à contraste élevé (craie blanche sur tableaux noirs, encre rouge sur papier blanc) pour qu'ils se détachent du décor.
3.  **Vérifiez l'année :** De nombreux coffres utilisent des dates de création (« Established ») trouvées sur les enseignes des magasins comme combinaisons.
4.  **Écoutez ISAC :** Votre IA embarquée annoncera souvent « Objet d'intérêt détecté » lorsque vous êtes à proximité d'un coffre ou d'un indice.

## FAQ

**Q : Est-ce que chaque coffre dans The Division Resurgence nécessite un code ?**

R : Non. La plupart des caisses de butin standard et des boîtes d'armes peuvent être ouvertes avec une simple interaction. Seuls les « Coffres verrouillés » trouvés dans des lieux nommés spécifiques nécessitent la saisie manuelle d'un code.

**Q : Puis-je partager le code de coffre-fort The Division Mobile avec mes coéquipiers ?**

R : Oui, les codes sont universels pour tous les joueurs. Si vous êtes dans un groupe, une seule personne a besoin de saisir le code pour que le coffre s'ouvre, et tout le monde à proximité recevra son propre butin individuel.

**Q : J'ai trouvé un coffre mais le code ne fonctionne pas. Que dois-je faire ?**

R : Vérifiez que vous êtes au bon endroit. De nombreux magasins se ressemblent sur la carte de NYC. Assurez-vous également de ne pas avoir confondu un « 9 » avec un « 8 » ou un « 7 » avec un « 1 », car les textures environnementales peuvent parfois être floues avec des réglages mobiles inférieurs.

**Q : Les récompenses des coffres s'adaptent-elles à mon niveau ?**

R : Oui. Bien que la rareté (ex: Supérieur) soit souvent fixe, les statistiques réelles et le niveau d'équipement (Gear Level) des objets à l'intérieur s'adapteront en fonction de votre niveau de personnage actuel et de votre Niveau de Monde au moment de l'ouverture.
