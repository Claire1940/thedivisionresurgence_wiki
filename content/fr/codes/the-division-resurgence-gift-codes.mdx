Les refuges offrent un sanctuaire, mais ils cachent aussi des secrets. En 2026, explorer chaque recoin de ces "zones vertes" est le meilleur moyen de trouver du butin de haut niveau sans combattre.

*   **Refuge Autumn's Hope :** Localisez le bureau de réception derrière le PNJ Mullins. Cherchez l'année d'établissement ("Established") sur le mur. Le code est **1883**.
*   **Refuge Dante's Run :** Sortez du refuge et dirigez-vous vers le mannequin dans le magasin de vêtements. Derrière le comptoir, vous trouverez un coffre. Le code, trouvé sur une affiche à proximité, est **7716**.
*   **Station Kerman :** Dirigez-vous vers les casiers près de la sortie. Le code est épinglé directement à côté des casiers : **7069**.

## Indices cryptiques avancés : Le cadeau d'anniversaire

L'un des coffres les plus complexes se situe près du Madison Field Hospital (Serotentia). Contrairement aux autres où le code est écrit clairement, celui-ci nécessite que vous trouviez d'abord un objet de collection.

1.  **Localisez l'objet de collection :** Cherchez un objet brillant sur le sol juste à l'extérieur de la salle du coffre.
2.  **Lisez l'indice :** L'objet mentionne un "Cadeau d'anniversaire" avec la date du "Mardi 11 mai".
3.  **Saisissez le code :** En utilisant le format de la date, le code est **1105**.

Ce coffre est nettement plus grand que les coffres standards et donne souvent des objets "Supérieurs" (Violets) ou même de "Haute qualité" (Or) selon votre niveau de palier mondial.

**Q : Pourquoi mon code de cadeau The Division Resurgence ne fonctionne-t-il pas ?**

R : Non, les coffres en jeu sont généralement des récompenses "uniques" par personnage. Une fois que vous avez ouvert le coffre à la colonie Spring Blossom ou à la station Kerman, il restera ouvert pour ce personnage.

**Q : Existe-t-il des codes permanents disponibles pour les nouveaux joueurs ?**

**Q : À quelle fréquence de nouveaux codes de cadeau sont-ils publiés ?**
