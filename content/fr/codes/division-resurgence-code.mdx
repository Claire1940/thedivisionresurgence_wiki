## Comprendre la mécanique des coffres dans Resurgence

Dans *The Division Resurgence*, les coffres servent de mécanisme de butin secondaire conçu pour encourager l'exploration approfondie du monde ouvert. Contrairement aux caisses de butin standard qui se réinitialisent ou contiennent des consommables aléatoires, ces coffres verrouillés sont souvent liés à des énigmes environnementales spécifiques ou à des « Indices Cryptiques » trouvés dans le menu des objets de collection. En 2026, il y a au moins 11 indices cryptiques majeurs à résoudre, chacun menant à une cache unique.

Lorsque vous approchez d'un coffre verrouillé, le jeu vous demandera une combinaison à trois ou quatre chiffres. Bien qu'il puisse être tentant de deviner, la solution se trouve toujours à proximité immédiate. Ces indices sont intégrés à la narration environnementale, faisant de la recherche un véritable travail de détective.

> **💡 Astuce :** Pulsez toujours votre environnement en entrant dans une nouvelle pièce. Les coffres et les documents contenant leurs codes brilleront souvent d'un contour blanc ou doré, ce qui les rend plus faciles à repérer dans les intérieurs sombres.

## Codes des coffres des colonies et de la base d'opérations

Les premiers coffres que vous rencontrerez se trouvent dans la colonie de Spring Blossom et dans la base d'opérations (BoO) principale de New York. Ils sont essentiels pour les nouveaux agents cherchant à remplacer leur équipement vert de départ par des objets bleus ou violets de qualité supérieure.

### Colonie de Spring Blossom
Après un voyage rapide vers la colonie de Spring Blossom, naviguez vers les niveaux supérieurs du bâtiment principal. Dans un petit bureau, vous trouverez un coffre caché dans un coin. L'indice pour ce **code division resurgence** est écrit clairement sur un tableau noir à proximité.

| Emplacement | Emplacement de l'indice | Code du coffre | Type de récompense |
| :--- | :--- | :--- | :--- |
| **Bureau de Spring Blossom** | Tableau noir à droite | **3580** | Objet supérieur + Ressources |

### Base d'opérations de NYC : Les dortoirs
À l'intérieur de la base d'opérations principale, spécifiquement dans l'aile des dortoirs où les agents se reposent, se trouve un panneau d'affichage rempli de divers papiers. Une note spécifique contient la combinaison du coffre situé à quelques pas de là.

| Emplacement | Emplacement de l'indice | Code du coffre | Type de récompense |
| :--- | :--- | :--- | :--- |
| **Dortoirs de la BoO** | Papier sur le panneau d'affichage | **1350** | Matériaux d'artisanat + Équipement |

## Emplacements des coffres du quartier commercial

Juste à l'extérieur de la base d'opérations, les rues environnantes contiennent plusieurs magasins de détail qui n'ont pas été entièrement pillés. Deux boutiques spécifiques — le magasin Oak Lock et un supermarché à proximité — abritent des coffres facilement accessibles si vous savez où regarder.

### Le magasin Oak Lock
Situé à droite de l'entrée de la base d'opérations, le magasin Oak Lock contient un coffre posé directement sur le sol. La combinaison est habilement cachée sur une enseigne promotionnelle à l'intérieur de la boutique.

### Le supermarché
En face du magasin Oak Lock se trouve un grand supermarché. Dirigez-vous vers le mur du fond pour trouver le coffre. Le code est affiché sur un grand panneau suspendu au-dessus de la section réfrigérée.

| Nom du magasin | Emplacement de l'indice | Code du coffre |
| :--- | :--- | :--- |
| **Oak Lock** | Enseigne de la vitrine | **122** |
| **Supermarché** | Panneau mural (Année) | **1987** |

## L'énigme du cadeau d'anniversaire de Serotentia

L'une des énigmes les plus complexes implique un grand coffre situé près de l'ancien Madison Field Hospital (un point de repère familier pour les vétérans du premier *Division* game). Ce coffre nécessite que vous trouviez d'abord un objet de collection pour comprendre la logique derrière le code.

En marchant dans la rue adjacente à l'hôpital, cherchez un objet de collection au sol intitulé « Indice de code : Cadeau d'anniversaire ». Le ramasser l'ajoute à votre menu d'Indices Cryptiques. Le texte mentionne un « Cadeau d'anniversaire » et liste des dates. Plus précisément, il fait référence au 5 novembre. En utilisant cette logique, le **code division resurgence** pour le grand coffre à l'intérieur du bâtiment Serotentia est dérivé de la date.

*   **Indice :** Cadeau d'anniversaire (mardi 5 novembre)
*   **Combinaison :** **1105**

> **Attention :** Les grands coffres comme celui de Serotentia déclenchent souvent des détections d'« Objet d'intérêt » pour les PNJ hostiles à proximité. Nettoyez la pièce avant de tenter de saisir le code pour éviter d'être interrompu par des Nettoyeurs ou des Émeutiers.

## Combinaisons des refuges : d'Autumn's Hope à Kerman Station

À mesure que vous progressez dans des districts de plus haut niveau, vous débloquerez des refuges. Ceux-ci servent de points de réapparition et de centres sociaux, mais ils contiennent également des trésors cachés pour les agents qui prennent la peine de fouiller les zones de réception et les arrière-salles.

### Autumn's Hope
Dans le refuge Autumn's Hope, cherchez près du PNJ Mullins. Derrière le bureau de réception, il y a un panneau indiquant l'année de fondation de l'établissement. Cette année sert de combinaison. Bien que la texture puisse paraître floue selon les paramètres graphiques de votre mobile, le code est le même pour tous les joueurs.

### Dante's Run
Sortez du refuge Dante's Run et dirigez-vous vers le magasin de vêtements attenant (cherchez le mannequin). Derrière la caisse principale se trouve un coffre. La combinaison est située sur une affiche colorée à l'avant de la boutique.

### Kerman Station
À Kerman Station, le coffre est en fait un ensemble de casiers verrouillés étiquetés « Safia Nerdies ». Le code est commodément épinglé sur le côté du casier sur un petit bout de papier.

| Refuge | Emplacement spécifique | Code |
| :--- | :--- | :--- |
| **Autumn's Hope** | Bureau de réception | **1883** |
| **Dante's Run** | Comptoir du magasin de vêtements | **7716** |
| **Kerman Station** | Zone des casiers | **7069** |

## Maximiser votre butin des coffres

Bien que le **code division resurgence** de chaque coffre soit statique, la qualité du butin peut varier légèrement en fonction de votre niveau actuel, bien que la plupart des coffres soient programmés pour fournir au moins un objet « Supérieur » (violet). Pour tirer le meilleur parti de ces récompenses, considérez les points suivants :

1.  **Espace d'inventaire :** Assurez-vous d'avoir au moins 5 à 10 emplacements libres avant une « tournée des coffres ». Vous ne voulez pas laisser de ressources précieuses au sol dans une zone hostile.
2.  **Menu des Indices Cryptiques :** Consultez régulièrement votre onglet « Objets de collection ». Si vous avez ramassé un indice mais n'avez pas trouvé le coffre, le menu fournira souvent un indice sur le district où se trouve le coffre.
3.  **Efficacité du voyage rapide :** La plupart des coffres sont situés à moins de 100 mètres d'un point de voyage rapide. Vous pouvez collecter tous les coffres mentionnés dans ce guide en moins de 20 minutes si vous utilisez la fonction de téléportation de la carte.

La recherche de chaque **code division resurgence** est un processus continu à mesure qu'Ubisoft ajoute plus de districts et de contenu saisonnier. Actuellement, le jeu suit 11 indices cryptiques majeurs, ce qui suggère qu'il y a encore plus de secrets cachés dans les coins sombres de la Dark Zone de New York et des quartiers environnants.

## FAQ

**Q : Les codes des coffres sont-ils les mêmes pour tous les joueurs dans Division Resurgence ?**

R : Oui, en 2026, les combinaisons de coffres sont des énigmes environnementales statiques. Cela signifie que le code trouvé sur un tableau noir dans votre jeu sera le même pour tout le monde, permettant aux agents de partager les solutions.

**Q : Puis-je ouvrir un coffre plus d'une fois ?**

R : Non, ces coffres du monde sont généralement des récompenses uniques par personnage. Une fois que vous avez saisi avec succès le **code division resurgence** et pillé le contenu, le coffre restera ouvert et ne pourra plus être pillé.

**Q : Que dois-je faire si le code ne fonctionne pas ?**

R : Vérifiez que vous êtes au bon endroit. Certains coffres se ressemblent mais appartiennent à des énigmes différentes. Assurez-vous également de saisir les chiffres dans le bon ordre ; par exemple, le coffre de Serotentia utilise le format Mois/Jour (1105).

**Q : Dois-je trouver l'indice avant de pouvoir saisir le code ?**

R : Dans la plupart des cas, vous pouvez saisir le code immédiatement si vous le connaissez déjà. Cependant, pour les coffres à « Indices Cryptiques » comme celui de Serotentia, il est fortement recommandé de ramasser l'objet de collection d'abord pour s'assurer que l'interaction avec le coffre est pleinement active et pour compléter votre galerie de collections.

Pour plus d'informations sur les mises à jour à venir et les nouveaux emplacements de coffres, visitez le [site officiel de Division Resurgence](https://www.ubisoft.com/en-us/game/the-division/resurgence) pour les dernières actualités de la communauté.
