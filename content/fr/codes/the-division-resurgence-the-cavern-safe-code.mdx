## Comprendre la mécanique des coffres en 2026

Les coffres-forts dans The Division Resurgence ne sont pas de simples objets décoratifs. Ils fonctionnent comme des mini-énigmes qui récompensent l'exploration. Contrairement aux caisses de butin standard qui se réinitialisent avec le temps, ces coffres verrouillés contiennent souvent des récompenses de haut niveau à usage unique, notamment des objets Supérieurs (violets) et des composants de fabrication essentiels.

Pour ouvrir un coffre, vous devez interagir avec le pavé numérique et saisir un code à trois ou quatre chiffres. Ces codes sont toujours cachés dans l'environnement immédiat. Les endroits courants pour chercher des indices incluent :

*   **Les tableaux noirs** à l'intérieur des colonies ou des salles de classe.
*   **Les tableaux d'affichage** dans les refuges ou les dortoirs.
*   **Les enseignes de magasins** qui indiquent des dates de création ou des numéros de téléphone.
*   **Les journaux audio et objets de collection** qui fournissent des « Indices de code » dans votre menu d'inventaire.

> **Avertissement :** Des ennemis patrouillent souvent dans les zones entourant ces coffres. Nettoyez toujours les environs immédiats des factions hostiles avant de tenter de saisir un code, car l'interface ne met pas le monde du jeu en pause.

## Guides détaillés des coffres

### Colonie Spring Blossom
Le coffre de Spring Blossom est l'un des premiers que vous rencontrerez probablement. Après un voyage rapide vers la colonie, dirigez-vous vers les niveaux supérieurs de la zone résidentielle. Vous trouverez une petite pièce contenant le coffre. Regardez immédiatement sur votre droite sur le tableau noir pour voir les chiffres **3580**. Ce coffre est connu pour donner des pièces d'armure Supérieures de début de jeu.

### Base d'Opérations : Dortoirs
À l'intérieur de la base d'opérations de NYC, dirigez-vous vers l'aile des dortoirs. Avant d'atteindre le coffre, il y a un tableau d'affichage encombré de papiers. Un prospectus spécifique indique clairement le code **1350**. Le saisir vous octroiera des ressources précieuses pour commencer vos premiers réétalonnages d'armes.

### L'énigme « Cadeau d'anniversaire » de Serotentia
Il s'agit d'un coffre unique situé près de l'ancien site du Madison Field Hospital. Contrairement aux autres, le code n'est pas écrit sur un mur. Vous devez ramasser un objet de collection au sol devant la tente. La note mentionne un « Cadeau d'anniversaire » le **5 novembre**. Dans le format de date américain utilisé par la logique du jeu (MM/JJ), cela se traduit par **1105**.

## Dépannage des codes de coffre

Si vous saisissez le bon code mais que le coffre reste verrouillé, envisagez les étapes suivantes :

1.  **Vérifiez les combats actifs :** Certains coffres sont verrouillés pendant des phases de mission spécifiques ou si des ennemis vous « détectent » dans la zone immédiate.
2.  **Vérifiez le format des dates :** Pour les coffres comme celui de Serotentia, assurez-vous de ne pas inverser le mois et le jour. Le jeu utilise principalement la logique MM/JJ pour ses énigmes.
3.  **Mettez à jour votre client :** Dans la version 2026 du jeu, certains codes ont été légèrement ajustés pour des événements saisonniers. Assurez-vous que votre jeu est à jour avec le dernier patch.
4.  **Bug d'interaction :** Si l'invite « Interagir » n'apparaît pas, essayez de faire un voyage rapide ailleurs puis de revenir au refuge le plus proche pour réinitialiser l'instance.

Pour les dernières mises à jour officielles sur les événements mondiaux et les notes de patch, visitez le [site officiel de Division Resurgence](https://www.ubisoft.com/en-us/game/the-division/resurgence).

## Stratégies d'exploration pour New York

Maximiser votre efficacité dans The Division Resurgence implique plus que de simplement connaître les codes. Vous devriez planifier vos « tournées de coffres » en parallèle de vos primes quotidiennes. Par exemple, les coffres de **Dante's Run** et **Autumn's Hope** peuvent être récupérés en quelques minutes l'un de l'autre grâce à leur proximité avec les points de voyage rapide.

Gardez toujours votre détecteur d'« Objets d'intérêt » actif. Votre système ISAC mettra souvent en évidence les coffres à travers les murs si vous êtes à moins de 15-20 mètres. C'est particulièrement utile dans les zones denses comme le **magasin Oak Lock** ou le **supermarché**, où les coffres sont cachés derrière des comptoirs ou dans des arrière-boutiques.

## FAQ

**Q : Est-ce que le code du coffre the cavern the division resurgence change selon les joueurs ?**

R : Non, les codes des coffres dans The Division Resurgence sont statiques pour tous les comptes. Une fois qu'un code est découvert pour un emplacement spécifique comme The Cavern, il reste le même pour tout le monde dans la version 2026 du jeu.

**Q : Puis-je ouvrir le même coffre plusieurs fois ?**

R : En général, ces coffres du monde sont des récompenses uniques par personnage. Une fois que vous avez saisi le code avec succès et récupéré le contenu, le coffre reste ouvert et ne peut plus être pillé.

**Q : Que se passe-t-il si je saisis un mauvais code trop de fois ?**

R : Il n'y a aucune pénalité en cas de mauvaise tentative. Vous pouvez essayer de saisir la combinaison autant de fois que nécessaire. Il n'y a pas de compte à rebours de verrouillage ni de mini-jeux de « piratage » associés à ces coffres environnementaux spécifiques.

**Q : Y a-t-il plus de 11 coffres dans le jeu ?**

R : À compter de la mise à jour d'avril 2026, il existe 11 coffres principaux d'« Indices cryptiques » dispersés dans Manhattan. Cependant, les développeurs ajoutent fréquemment de nouveaux coffres lors des mises à jour saisonnières, il est donc sage de vérifier les tableaux d'affichage dans les nouveaux refuges au fur et à mesure de leur sortie.
