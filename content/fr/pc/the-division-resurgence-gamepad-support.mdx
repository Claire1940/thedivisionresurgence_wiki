## Comment connecter votre manette

La configuration de la **prise en charge des manettes dans The Division Resurgence** est un processus simple, mais il varie légèrement selon votre système d'exploitation mobile. Assurez-vous que votre manette est complètement chargée avant de tenter de l'appairer pour éviter toute déconnexion pendant les missions critiques.

### Configuration sur Android
1. Ouvrez les **Paramètres** de votre appareil et accédez à **Bluetooth**.
2. Mettez votre manette en mode appairage (ex : maintenez les boutons Share et PS sur une DualSense).
3. Sélectionnez la manette dans la liste des appareils disponibles.
4. Lancez The Division Resurgence ; le jeu vous proposera un tutoriel pour manette dès l'entrée.

### Configuration sur iOS
1. Allez dans **Réglages** > **Bluetooth** et assurez-vous qu'il est activé.
2. Activez le mode appairage sur votre manette.
3. Appuyez sur le nom de l'appareil sous "Autres appareils".
4. Une fois que "Connecté" apparaît, ouvrez le jeu. Vous pouvez vérifier la connexion dans le menu des paramètres du jeu.

## Configuration des touches et commandes par défaut

La disposition par défaut pour la **prise en charge des manettes dans The Division Resurgence** suit les standards de l'industrie pour les jeux de tir à la troisième personne. Si vous avez joué aux précédents opus de la série, vous ne serez pas dépaysé. Le jeu utilise presque tous les boutons d'une manette standard pour gérer les systèmes complexes du RPG.

| Action | Entrée Xbox | Entrée PlayStation |
|---|---|---|
| **Déplacement / Sprint** | Stick gauche (Clic pour Sprinter) | L3 |
| **Visée / Caméra** | Stick droit (Clic pour Corps à corps) | R3 |
| **Tirer / Attaquer** | RT | R2 |
| **Viser (ADS)** | LT | L2 |
| **Se mettre à couvert / Franchir** | A | Croix |
| **Recharger / Interagir** | X | Carré |
| **Changer d'arme** | Y | Triangle |
| **Utiliser Compétence 1** | LB | L1 |
| **Utiliser Compétence 2** | RB | R1 |

> **⚠️ Attention :** Certaines manettes Bluetooth plus anciennes peuvent rencontrer des problèmes de "zone morte" où le personnage bouge légèrement même lorsque le stick est au neutre. Vous pouvez ajuster les paramètres de Zone Morte dans l'onglet Gameplay du menu des options.

## Optimisation des performances à la manette pour les spécialisations

Différentes classes dans The Division Resurgence bénéficient de réglages spécifiques à la manette. Comme le jeu propose des rôles variés comme le Démolisseur et l'Avant-garde, vos paramètres de sensibilité doivent refléter votre style de jeu.

### Démolisseur (Explosifs puissants)
Le Démolisseur compte sur le placement précis des grenades et des tirs d'armes de signature. Pour cette classe, une "Sensibilité ADS" légèrement inférieure est recommandée pour garantir que les arcs de vos grenades soient parfaitement placés. Comme vous utiliserez fréquemment L1/R1 pour vos mines téléguidées et vos tourelles, assurez-vous que vos gâchettes hautes soient réactives.

### Avant-garde (Soutien et Tank)
En tant qu'Avant-garde, votre objectif principal est le positionnement et la gestion du bouclier. La **prise en charge des manettes dans The Division Resurgence** vous permet de vous déplacer plus fluidement lorsque votre bouclier est déployé par rapport aux commandes tactiles. Vous pourriez vouloir augmenter votre "Sensibilité de rotation" pour réagir rapidement aux ennemis qui tentent de vous contourner.

### Médecin de terrain (Soigneur)
Le soin nécessite un changement de cible rapide. Utiliser la croix directionnelle (D-Pad) pour faire défiler les coéquipiers est souvent plus rapide que d'appuyer sur de petites icônes sur un écran. Assurez-vous que votre "Sensibilité de l'interface" est suffisamment élevée pour naviguer sur la carte tactique et les menus sociaux pendant les accalmies au combat.

## Résolution des problèmes courants liés aux manettes

Si vous constatez que la **prise en charge des manettes dans The Division Resurgence** ne fonctionne pas correctement, suivez ces correctifs courants pour reprendre l'action :

1. **Boutons ne répondant pas :** Cela arrive souvent si la manette a été connectée *après* le lancement du jeu. Fermez complètement l'application, assurez-vous que la manette est appairée dans l'OS de votre téléphone, puis redémarrez le jeu.
2. **Mauvaises icônes de touches :** Si vous utilisez une manette PlayStation mais voyez des icônes Xbox, vérifiez le paramètre "Type de manette" dans le menu. Dans certaines versions bêta, cela peut être réglé par défaut sur les icônes Xbox.
3. **Ghosting/Latence :** C'est généralement causé par des interférences avec d'autres appareils Bluetooth. Essayez d'éteindre le Bluetooth sur les tablettes ou ordinateurs à proximité pour libérer le signal.
4. **Mises à jour du firmware :** Les manettes modernes comme la DualSense et les manettes Xbox Series nécessitent des mises à jour du firmware. Connectez-les à une console ou un PC pour vous assurer qu'elles exécutent le dernier logiciel pour une compatibilité mobile maximale.

## FAQ

**Q : The Division Resurgence prend-il en charge les manettes pour toutes les classes ?**

R : Oui, la **prise en charge des manettes dans The Division Resurgence** est universelle pour toutes les spécialisations, y compris le Démolisseur, le Rempart, le Médecin de terrain et l'Avant-garde. La configuration des touches reste cohérente, bien que les compétences spécifiques aux classes soient assignées aux gâchettes hautes (L1/R1 ou LB/RB).

**Q : Puis-je personnaliser la configuration des touches ?**

R : Bien qu'il existe plusieurs configurations prédéfinies, le jeu permet également un remapping limité des touches dans les paramètres. Vous pouvez échanger les fonctions de certains boutons pour mieux les adapter à votre confort, comme l'inversion des commandes "Couverture" et "Sprint".

**Q : Y a-t-il une aide à la visée lors de l'utilisation d'une manette ?**

R : Oui, le jeu propose un niveau d'aide à la visée ajustable pour les utilisateurs de manettes. Cela aide à compenser les différences inhérentes entre un stick analogique et une souris, rendant l'expérience fluide et gratifiante sans pour autant jouer à votre place.

**Q : L'utilisation d'une manette me placera-t-elle dans des files d'attente séparées ?**

R : En 2026, Ubisoft a mis en place le cross-play entre les utilisateurs tactiles et manettes dans la plupart des modes pour maintenir des temps d'attente courts. Cependant, certains paliers compétitifs de haut niveau peuvent proposer un matchmaking basé sur le type d'entrée pour garantir l'équité.
