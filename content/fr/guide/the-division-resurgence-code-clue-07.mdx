## Comprendre les mécaniques de coffres et les indices

Dans *The Division Resurgence*, les coffres sont classés comme des « Objets d'intérêt ». Lorsque vous êtes à proximité de l'un d'eux, votre système ISAC annonce généralement : « Objet d'intérêt détecté ». Cependant, interagir avec le coffre n'est que la moitié de la bataille. Chaque coffre nécessite une combinaison à trois ou quatre chiffres.

Ces combinaisons ne sont jamais aléatoires ; elles sont intégrées au monde du jeu. Vous pouvez les trouver sur :
*   **Les Tableaux noirs :** Souvent trouvés dans les colonies ou les salles de classe.
*   **Les Affiches et panneaux :** Vérifiez les vitrines des magasins ou les dates de fondation (« Established ») sur les façades des bâtiments.
*   **Les Objets de collection :** Plus précisément, le menu « Indices cryptiques » suit des éléments comme l'indice de code 07 de The Division Resurgence, qui fournit une énigme ou une référence directe à une date ou un événement.

> **Attention :** Certains coffres sont gardés par des ennemis d'élite ou nécessitent que vous terminiez un « Événement mondial » spécifique avant que la pièce ne devienne accessible. Sécurisez toujours le périmètre avant de tenter de saisir un code.

## Guide détaillé : L'indice de code 07 de The Division Resurgence

Alors que de nombreux coffres utilisent des indices visuels simples, la série « Indices cryptiques » — incluant **l'indice de code 07 de The Division Resurgence** — nécessite un peu plus d'investigation. Ceux-ci sont souvent liés à la narration globale des efforts de la SHD pour stabiliser Manhattan.

Lorsque vous ramassez un indice cryptique, il est stocké dans votre onglet « Collection ». Pour l'indice 07, les joueurs le trouvent souvent à proximité du district financier ou lors de missions secondaires spécifiques impliquant des restes de la JTF. L'indice fait généralement référence à une date importante ou à un numéro de chambre spécifique trouvé dans les entrées du lore.

### Comment résoudre les énigmes cryptiques
1.  **Ouvrez le menu Inventaire :** Naviguez vers la section « Objets de collection ».
2.  **Sélectionnez Indices cryptiques :** Cherchez l'Indice 07 ou son équivalent (comme l'indice « Cadeau d'anniversaire »).
3.  **Analysez le texte :** Si l'indice indique « Mardi 11 mai », le code est probablement **1105** ou **0511**.
4.  **Vérifiez les dates environnementales :** Pour les coffres dans les bâtiments historiques, cherchez un panneau « Established [Année] ». Si l'année est 1883, c'est votre combinaison.

## Coffres cachés de la Base d'Opérations

La Base d'opérations (BoO) est plus qu'un simple centre social ; elle contient plusieurs coffres que les débutants négligent souvent. Ils sont excellents pour prendre de l'avance sur votre build avant de vous aventurer dans la Dark Zone.

### Le coffre du dortoir
Situé dans l'aile résidentielle où les agents apparaissent après un voyage rapide. Cherchez le tableau d'affichage avant d'entrer dans les quartiers de sommeil. Un petit bout de papier contient le code **1350**. Ce coffre est crucial car il laisse souvent tomber des schémas pour les mods d'armes de début de jeu.

### La boucle de Madison Street
Directement à l'extérieur de la BoO, si vous prenez à gauche vers l'hôpital de campagne de Madison (un clin d'œil au jeu original), vous trouverez une série de vitrines.
*   **Le coffre de Serotentia :** Il s'agit d'un grand coffre au sol. Pour l'ouvrir, vous devez d'abord ramasser l'objet de collection « Cadeau d'anniversaire » sur le sol devant la tente. L'indice mentionne le 5 novembre, ce qui donne le code **1105**.
*   **Le supermarché :** Situé en face du magasin Oaklock. Le code **1987** est visible sur un grand panneau rouge au-dessus de la section des produits surgelés.

> **💡 Conseil :** Améliorez toujours votre compétence « Récupération » dans votre arbre de spécialisation pour augmenter la qualité du butin obtenu dans ces coffres du monde ouvert.

## Stratégie d'exploration pour 2026

À mesure que le jeu évolue en 2026, de nouveaux districts sont ajoutés, chacun avec son propre ensemble de coffres et d'indices cryptiques. Pour garder une longueur d'avance, suivez cette liste de contrôle d'exploration :

1.  **Voyage rapide vers les planques :** Chaque planque (Autumn's Hope, Dante's Run, etc.) possède généralement un coffre caché à moins de 100 mètres de l'entrée.
2.  **Cherchez les tableaux noirs :** Dans toute pièce ressemblant à un bureau ou une salle de classe, vérifiez les tableaux. Les développeurs cachent fréquemment des codes à la vue de tous en utilisant des textures de « craie blanche ».
3.  **Interagissez avec chaque objet de collection :** Même si vous n'êtes pas un complétiste, les objets de collection comme l'indice de code 07 de The Division Resurgence sont le seul moyen de déverrouiller les « Coffres Maîtres » de haute valeur qui contiennent des composants exotiques.
4.  **Utilisez un casque :** Le signal sonore ISAC pour un « Objet d'intérêt » est directionnel. Si vous entendez le signal, le coffre est probablement derrière un bureau ou dans un petit placard à proximité.

Pour plus d'informations sur les événements saisonniers à venir et les ensembles d'équipement, visitez le [site officiel de The Division Resurgence](https://www.ubisoft.com/en-us/game/the-division/resurgence) pour les dernières mises à jour des développeurs.

## FAQ

**Q : Où puis-je trouver l'indice de code 07 de The Division Resurgence ?**

R : Cet indice fait partie de la collection d'indices cryptiques. On le trouve généralement dans les districts avancés du jeu, souvent à l'intérieur d'immeubles de bureaux ou de stations de relais SHD abandonnées. Le ramasser fournira l'indice nécessaire pour les coffres de haute sécurité à proximité.

**Q : Les codes de coffres changent-ils chaque semaine ?**

R : Non, les codes de coffres dans *The Division Resurgence* sont statiques et ne changent pas avec les réinitialisations hebdomadaires. Une fois que vous connaissez un code, il fonctionnera pour chaque personnage de votre compte pendant toute la durée de la saison 2026.

**Q : Pourquoi ne puis-je pas interagir avec un coffre même si je connais le code ?**

R : Certains coffres sont verrouillés derrière la progression de mission. Si vous n'avez pas atteint un certain niveau ou nettoyé le « Point de contrôle » local, l'invite d'interaction peut ne pas apparaître. Assurez-vous d'abord que la zone est « verte » (sécurisée) sur votre carte.

**Q : Y a-t-il des objets exotiques dans ces coffres ?**

R : Bien que les objets exotiques soient extrêmement rares dans les coffres standards du monde, les « Coffres Maîtres » liés à de longues chaînes d'indices cryptiques ont un faible pourcentage de chance de donner des composants exotiques utilisés pour fabriquer des armes de haut niveau.
