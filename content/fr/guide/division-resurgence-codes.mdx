## Comment utiliser les codes Division Resurgence pour les coffres verrouillés

Les coffres verrouillés dans ce jeu fonctionnent comme des énigmes environnementales. Contrairement aux codes promotionnels échangeables que vous saisissez dans un menu, ces codes se trouvent par l'exploration. Lorsque vous rencontrez un coffre, interagir avec lui fera apparaître un clavier numérique. En saisissant la combinaison correcte à trois ou quatre chiffres, le coffre s'ouvrira, vous récompensant avec un butin adapté à la difficulté de la zone.

La plupart des codes sont cachés à la vue de tous : sur des tableaux noirs, des affiches ou même dans des journaux audio à collectionner. Vous trouverez ci-dessous un tableau de référence rapide pour les coffres les plus courants trouvés pendant la progression du début au milieu de jeu en 2026.

## Emplacements détaillés des coffres et combinaisons

Pour vous aider à les localiser, nous avons détaillé les chemins spécifiques vers chaque coffre. L'efficacité est primordiale lors de la collecte d'équipement, utilisez donc les points de voyage rapide listés ci-dessous pour gagner du temps.

### Coffre du campement Spring Blossom
C'est probablement le premier coffre que la plupart des agents rencontreront. Après un voyage rapide vers Spring Blossom Settlement, dirigez-vous vers les niveaux supérieurs de la zone résidentielle. Suivez les escaliers jusqu'au dernier étage jusqu'à entrer dans une petite pièce contenant le coffre.
*   **Le Code :** 3580
*   **Où trouver l'indice :** Regardez le tableau noir immédiatement à droite du coffre. Les chiffres sont écrits clairement à la craie blanche.

### Base d'opérations NYC : Les dortoirs
À l'intérieur de votre hub principal, un coffre est caché dans les quartiers d'habitation. Entrez dans les dortoirs et cherchez le panneau d'affichage central où les agents posent des flyers.
*   **Le Code :** 1350
*   **Où trouver l'indice :** Un petit morceau de papier épinglé sur le panneau d'affichage contient la séquence de quatre chiffres.

### Magasin Oaklock et Supermarché
Ces deux coffres sont situés très près l'un de l'autre, juste en bas de la rue de l'entrée principale de la base d'opérations. Prenez à droite en sortant de l'entrée et descendez le pâté de maisons.
1.  **Magasin Oaklock :** Situé sur le côté gauche de la rue. Le coffre est au sol. Utilisez le code **122**, qui se trouve sur l'enseigne du magasin.
2.  **Supermarché :** Directement en face d'Oaklock. Le coffre se trouve le long du mur du fond. Utilisez le code **1987**, qui fait référence à l'année de fondation du magasin, trouvée sur le mur au-dessus du coffre.

### Le coffre "Cadeau d'anniversaire" de Serotentia
Ce coffre est unique car il nécessite un objet de collection pour être résolu. Il est situé près du site du Madison Field Hospital. Cherchez un objet de collection au sol devant l'entrée de Serotentia étiqueté "Indice de code : Cadeau d'anniversaire".
*   **Le Code :** 1105
*   **La Logique :** L'indice mentionne un anniversaire le 5 novembre (11/05). Saisir cela dans le grand coffre au sol débloquera la récompense.

## Codes des planques : Autumn's Hope et Dante's Run

À mesure que vous débloquez de nouvelles planques sur la carte, vous constaterez que beaucoup d'entre elles servent de mini-hubs avec leurs propres secrets cachés. Ce sont souvent les **codes division resurgence** les plus faciles à manquer car les agents ont tendance à voyager rapidement et à quitter immédiatement la zone.

### Autumn's Hope
À l'intérieur de la planque Autumn's Hope, dirigez-vous vers le bureau de réception. Le coffre est dissimulé près du PNJ Mullins.
*   **Le Code :** 1883
*   **L'Indice :** Regardez l'année "Established" sur le panneau derrière le bureau. Même si vos paramètres graphiques la rendent floue, 1883 est la combinaison confirmée.

### Dante's Run
Sortez de la planque Dante's Run par la porte principale et prenez à droite vers la zone du magasin de vêtements (cherchez les mannequins). Le coffre est situé derrière le comptoir de caisse.
*   **Le Code :** 7716
*   **L'Indice :** Une affiche lumineuse sur le devant du bureau affiche clairement les chiffres.

### Station Kerman
À la station Kerman, sortez de la salle d'apparition principale et cherchez les casiers "Safia Nerdies".
*   **Le Code :** 7069
*   **L'Indice :** Une petite note est épinglée directement sur le côté du bloc de casiers contenant le coffre.

| Planque | Détail de l'emplacement | Code |
| :--- | :--- | :--- |
| **Autumn's Hope** | Derrière le bureau de réception | **1883** |
| **Dante's Run** | Derrière le comptoir du magasin | **7716** |
| **Kerman Station** | À côté des casiers | **7069** |

## Dépannage et conseils pour trouver des codes

Si vous rencontrez un coffre non répertorié ici, ne vous inquiétez pas. Le jeu propose 11 "Indices Cryptiques" dans le menu des objets de collection qui pointent vers d'autres caches cachées. Pour les trouver par vous-même, gardez ces conseils à l'esprit :

1.  **Observez l'environnement :** La plupart des codes se trouvent à moins de 10 mètres virtuels du coffre. Regardez les affiches, les tableaux noirs et les enseignes au néon.
2.  **Ramassez tout :** Certains coffres ne permettent pas d'interaction tant que vous n'avez pas ramassé l'objet de collection "indice" correspondant dans la zone proche.
3.  **Écoutez votre ISAC :** Votre IA de bord annoncera souvent "Objet d'intérêt détecté" lorsque vous êtes à proximité d'un coffre verrouillé ou d'un indice.
4.  **Paramètres de luminosité :** Certains codes sont écrits à l'encre sombre sur des surfaces sombres. Si vous avez du mal à voir un panneau, augmentez momentanément la luminosité du jeu.

> **Attention :** Soyez prudent lorsque vous saisissez des codes dans les "Zones Contaminées". Vos filtres ont une durée limitée, et passer trop de temps à essayer de deviner un code peut entraîner une mort rapide. Ayez toujours le code prêt avant d'entrer.

Pour plus d'informations sur les mises à jour officielles et les événements saisonniers, visitez le [site officiel de Division Resurgence](https://www.ubisoft.com/en-us/game/the-division/resurgence) pour rester informé des derniers correctifs.

## FAQ

**Q : Les codes Division Resurgence sont-ils les mêmes pour tous les joueurs ?**

R : Oui, les combinaisons de coffres sont des énigmes environnementales statiques. Elles ne changent pas entre les comptes ou les sessions, ce qui signifie que vous pouvez utiliser les mêmes codes répertoriés dans ce guide pour débloquer des récompenses sur n'importe quel personnage.

**Q : Ces coffres réapparaissent-ils après les avoir ouverts ?**

R : Généralement, ces coffres verrouillés spécifiques sont des récompenses uniques par personnage. Cependant, ils offrent un boost significatif à vos ressources et contiennent souvent des objets de niveau "Supérieur" (violet) très utiles pour monter en niveau.

**Q : Puis-je trouver des codes promo pour des objets gratuits dans Division Resurgence ?**

R : Bien que ce guide se concentre sur les codes de coffres en jeu, Ubisoft publie occasionnellement des codes promo échangeables lors d'événements saisonniers ou via les réseaux sociaux. Ceux-ci sont généralement saisis dans le menu "Paramètres" ou "Boutique" sous un onglet "Échanger le code".

**Q : Que faire si un code ne fonctionne pas ?**

R : Tout d'abord, assurez-vous d'être au bon emplacement de coffre. Certains coffres se ressemblent mais appartiennent à des districts différents. Deuxièmement, vérifiez si vous avez ramassé tous les objets de collection à proximité, car certains coffres nécessitent que l'"Indice" soit dans votre inventaire avant que le clavier ne devienne actif.
