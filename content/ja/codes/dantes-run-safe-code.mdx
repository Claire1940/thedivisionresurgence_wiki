## ダンテズ・ランの金庫コードの見つけ方

ダンテズ・ラン・セーフハウスは、ミッドタウン地区で活動するエージェントにとって重要な拠点です。この場所に関連する金庫を見つけるには、まずダンテズ・ラン・セーフハウスにファストトラベルしてください。スポーンしたら、マネキンのディスプレイの近くにある出口ドアを探します。

このドアから出ると、すぐ隣の衣料品店に直接つながっています。店の奥へ進み、メインのサービスデスクの後ろを確認してください。床に置かれた小さな鍵のかかった金庫が見つかります。ゲーム内では手がかりを探すよう促されますが、デスクの前面にあるポスターを見ることで時間を節約できます。探すべき場所を知っている人には、コードがはっきりとそこに記されています。

| 場所 | 金庫の位置 | 手がかりの場所 | コード |
| :--- | :--- | :--- | :--- |
| **ダンテズ・ラン** | 衣料品店のデスク裏 | デスク前面のポスター | **7716** |

> **💡 ヒント:** コードを入力しようとする前に、必ず周囲の徘徊している敵を排除してください。インタラクトメニューを開いている間は、近くの勢力からの奇襲に対して無防備になります。

## 金庫の場所とコードのマスターリスト

ダンテズ・ランは、生存者が貴重品を隠した多くの場所の一つに過ぎません。2026年の様々な居住地やセーフハウスにおいて、エージェントは他にもいくつかのロックされたコンテナを発見しています。各金庫には通常、「最上級」（紫）ティアのアイテムと、不可欠なクラフト資源や通貨が含まれています。

### セーフハウスのコンテナ

セーフハウスは単なるリスポーン地点ではありません。そこは宝の山です。開発者はほぼすべての主要なセーフハウスに金庫を隠しており、多くの場合、ちょっとした環境パズルを解いたり、特定の収集品を見つけて組み合わせを明らかにする必要があります。

1.  **オータムズ・ホープ:** 受付デスクの近くにあります。コードは、マリンズの後ろにある看板の「設立（Established）」年に由来します（**1883**）。
2.  **カーマン・ステーション:** メインルームを出る際のロッカーの近くにあります。コードは近くの壁に貼られています（**7069**）。
3.  **セロテンティア:** これは「誕生日の贈り物（Birthday Gift）」というタイトルの収集品の手がかりを必要とする大きめの金庫です。手がかりには11月5日と言及されており、コード（**1105**）に繋がります。

## 金庫の報酬とルートティアの理解

『ディビジョン リサージェンス』で金庫を開けることは、単なる一時の快感ではありません。それはギアの最適化のための戦略的な行動です。ほとんどの金庫は、現在のレベルに応じた少なくとも1つの装備をドロップするように設定されており、通常のクレートで見つかるものよりも希少度が高いことが多いです。

### 一般的なドロップ品

**ダンテズ・ランの金庫コード**やその他の組み合わせを正しく入力すると、以下の報酬が期待できます：

*   **最上級装備:** 通常、2つ以上のアトリビュートスロットを持つ武器またはアーマー。
*   **貴重な資源:** 作戦基地でのクラフトに使用される電子部品、布、武器パーツなど。
*   **Eクレジット:** 設計図やベンダーアイテムを購入するための主要通貨。
*   **XPブースト:** エージェントレベル全体に寄与する少量の経験値。

| 報酬タイプ | 希少度 | 頻度 |
| :--- | :--- | :--- |
| **最上級武器** | 紫 | **高** |
| **優秀装備** | 金 | **低** |
| **クラフト材料** | 標準 | **確定** |

## コード探しのエキスパートチップ

このガイドに記載されていない金庫に遭遇した場合、自分で組み合わせを見つけるためのいくつかのコツがあります。ディビジョンのエージェントは周囲を観察するように訓練されており、ゲームはその細部への注意に報いてくれます。

> **⚠️ 警告:** 一部の金庫は特定のサイドミッションに関連付けられています。金庫が反応しない場合は、ミッションログをチェックして、特定の目的を先に有効にする必要があるか確認してください。

*   **壁をチェックする:** ほとんどのコードは、金庫から10〜15メートル以内の黒板、ホワイトボード、またはポスターに書かれています。
*   **収集品を読む:** セロテンティアの金庫のように、先に「電子デバイス」や「メモ」の収集品を拾う必要があるものもあります。これらは「謎の手がかり」メニューに表示されます。
*   **ISACの指示を聞く:** ロックされた金庫や手がかりの近くにいると、搭載AIが「関心のあるオブジェクトを検知（Object of interest detected）」とアナウンスすることがよくあります。
*   **西暦を探す:** 多くのコードは4桁の西暦（1987年や1883年など）です。「Founded in（設立）」や「Established」の看板を見つけたら、まずその数字を試してみてください。

公式アップデートやゲームメカニクスの詳細については、[Ubisoft ディビジョン リサージェンス公式サイト](https://www.ubisoft.com/en-us/game/the-division/resurgence)をご覧ください。

## 2026年におけるギアスコアの重要性

キャンペーンを進めるにつれて、ギアスコアはエンドゲームコンテンツへの門番となります。早い段階で**ダンテズ・ランの金庫コード**を利用することで、レベル5の白ティア装備でレベル10〜15の敵に苦戦することを避けられます。これらの金庫は紫アイテムを確定で提供するため、「緑装備」の期間をスキップする最も効率的な方法です。

現在の2026年のメタでは、早期に高レアリティの装備を揃えることで、ワールドティアをスムーズに進み、ダークゾーンやコンフリクトモードをより早く解放できます。新しい地区に入ったら、必ずセーフハウスの隠されたコンテナに立ち寄るようにしましょう。

## よくある質問（FAQ）

**Q: ダンテズ・ランの金庫コードはプレイヤーごとに異なりますか？**

A: いいえ、『ディビジョン リサージェンス』のコードは固定です。**ダンテズ・ランの金庫コード**は、キャラクターレベルやプラットフォームに関係なく、常に7716です。

**Q: これらの金庫は何度も開けることができますか？**

A: いいえ、これらはキャラクターごとに1回限りの報酬です。一度金庫を開けて戦利品を受け取ると、その金庫は開いたままになり、再び略奪することはできません。

**Q: コード7716が機能しない場合はどうすればよいですか？**

A: 正しい場所にいるか確認してください。ニューヨークには複数の衣料品店がありますが、ダンテズ・ラン・セーフハウスに直結している（マネキン出口の近く）店舗のみがこの特定のコードを受け付けます。

**Q: ゲーム内には11個以上の金庫がありますか？**

A: 2026年の最新アップデート時点では、主要な「謎の手がかり」金庫は11個ありますが、プレイヤーはオープンワールドでさらに小さな、マークされていないコンテナを見つけ続けています。インタラクト可能なキーパッドには常に目を光らせておきましょう！
