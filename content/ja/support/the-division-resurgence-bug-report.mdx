## 「ディビジョン リサージェンス」のバグ報告方法

Ubisoftは、モバイルプレイヤーからのフィードバックが適切なエンジニアリングチームに届くよう、プロセスを合理化しました。問題を報告する方法は主に2つあります。ゲーム内のサポートメニューから、または公式のUbisoftモバイルサポートポータル経由です。ゲーム内からの方法は、開発者がエラーの原因を特定するのに役立つ特定のセッションログを自動的に添付できるため、一般的に推奨されます。

ゲーム内で報告を提出するには、**設定**メニューに移動し、**サポート**タブを選択して、「バグを報告」オプションを探してください。メニューに到達する前にゲームがクラッシュする場合は、[Ubisoft公式ヘルプウェブサイト](https://www.ubisoft.com/en-us/help)にアクセスして、手動でチケットを提出する必要があります。

| 方法 | 最適な用途 | 対応速度 |
| :--- | :--- | :--- |
| **ゲーム内メニュー** | 視覚的グリッチ、UIエラー、ミッションのバグ | **速い** |
| **ウェブポータル** | ログイン問題、起動時のクラッシュ、アカウント問題 | **中程度** |
| **公式Discord** | 一般的なコミュニティフィードバックと既知の問題 | **様々** |

## 報告に必要な情報

**ディビジョン リサージェンスのバグ報告**を作成する際、提供する情報の質は、バグが修正されるまでの速度に直接影響します。「ゲームがクラッシュした」という漠然とした報告では、QAチームにとってほとんど役に立ちません。代わりに、開発者が問題を再現できるように、構造化された形式に従ってください。

> **💡 ヒント：** バグが発生している短い画面録画を常に試みてください。視覚的な証拠は、UIやアニメーションのエラーを特定する上で、書かれた説明よりもはるかに価値があります。

### 必須詳細チェックリスト

| 要件 | 説明 |
| :--- | :--- |
| **デバイスモデル** | 使用している正確なスマートフォンまたはタブレット（例：iPhone 15 Pro、Samsung S24）。 |
| **OSバージョン** | 現在のオペレーティングシステムバージョン（iOS 19またはAndroid 16）。 |
| **再現手順** | バグが発生する直前に行ったことを正確に記した番号付きリスト。 |
| **期待される動作** | 何が起こるべきだったか（例：「ドアが開くはずだった」）。 |
| **実際の動作** | 実際に何が起こったか（例：「ドアはロックされたままで、インタラクションに失敗した」）。 |

## バグの重大度分類

すべてのバグが同じではありません。**ディビジョン リサージェンスのバグ報告**を提出する際、問題の重大度を分類するよう求められる場合があります。これにより、Ubisoftのトリアージチームは、ゲームを停止させる「ブロッカー」バグを、軽微な「コスメティック」な問題よりも優先して処理できます。以下の表を参考に、あなたの問題がどのカテゴリに当てはまるか判断してください。

| 重大度レベル | ゲームプレイへの影響 | 優先度 |
| :--- | :--- | :--- |
| **ブロッカー** | ゲームがホーム画面にクラッシュするか、ミッションの完了を妨げる。 | **致命的** |
| **メジャー** | 著しいゲームプレイへの影響（例：スキルがダメージを与えない）。 | **高い** |
| **マイナー** | 機能するが煩わしい（例：一時的に音が途切れる）。 | **中程度** |
| **コスメティック** | 視覚的な問題のみ（例：テクスチャのちらつき、クリッピング）。 | **低い** |

## 報告前のトラブルシューティング

**ディビジョン リサージェンスのバグ報告**に時間を費やす前に、いくつかの標準的なトラブルシューティング手順を実行する価値があります。モバイルデバイスでの多くの「バグ」は、実際にはローカルキャッシュの破損、ストレージ不足、または不安定なネットワーク接続が原因であることがあります。

1.  **アプリケーションの再起動：** タスクスイッチャーからアプリを完全に終了し、再起動してください。
2.  **キャッシュのクリア（Androidのみ）：** デバイスの設定に移動し、アプリを見つけてキャッシュをクリアしてください（アカウントがリンクされていない限り、データをクリアしないでください）。
3.  **アップデートの確認：** ゲームとデバイスのOSの両方が2026年の最新バージョンに更新されていることを確認してください。
4.  **ネットワークの確認：** Wi-Fiからモバイルデータに切り替える（またはその逆）ことで、接続固有のグリッチではないことを確認してください。
5.  **ストレージの空き容量を確保：** ゲームが一時ファイルやシェーダーを処理するために、少なくとも5GBの空き容量があることを確認してください。

これらの手順で問題が解決しない場合は、上記で述べた公式チャネルを通じて**ディビジョン リサージェンスのバグ報告**を提出してください。

## 追跡とフォローアップ

報告を提出した後、Ubisoftモバイルサポートサイトの「マイケース」または「報告済みの問題」セクションを通じて、通常その進捗状況を追跡できます。軽微なバグの場合、個別の返信を受け取れないこともありますが、データは将来のパッチのために収集されます。アカウントの紛失や通貨のエラーなどの重大な問題については、サポートチケットで積極的に対応し、エージェントから要求された追加情報を提供してください。

開発チームは、公式フォーラムやDiscordで「既知の問題」リストを頻繁に公開しています。報告する前にこれらのリストを確認し、開発者がすでにあなたの特定の問題の修正に取り組んでいるかどうかを確認することをお勧めします。これにより、あなたとサポートチームの両方の貴重な時間を節約できます。

## 2026年における一般的なエラーコード

バグに特定のエラーコードが含まれる場合は、**ディビジョン リサージェンスのバグ報告**の件名にそのコードを含めるようにしてください。これらのコードは、多くの場合、特定のサーバー側またはクライアント側の障害を示しています。

| エラーコード | 可能性のある意味 | 推奨されるアクション |
| :--- | :--- | :--- |
| **DELTA-01** | 一般的な接続障害 | サーバー状況とローカルインターネット接続を確認してください。 |
| **MIKE-01** | アカウント/プロファイルの読み込みエラー | 10分待ってから再度ログインを試みてください。 |
| **ECHO-02** | サーバー同期の問題 | ゲームのバージョンが最新であることを確認してください。 |
| **BRAVO-04** | 認証エラー | Ubisoft Connectアカウントに再度ログインしてください。 |

## よくある質問

**Q: 「ディビジョン リサージェンス」のバグ報告を提出するのに最適な場所はどこですか？**

A: 最も効果的な方法は、設定タブにあるゲーム内の「サポート」メニューからです。この方法では、デバイスのログが開発者に直接送信されます。ゲームに入れないような問題の場合は、Ubisoftモバイルサポートウェブサイトを利用してください。

**Q: バグを報告すると報酬がもらえますか？**

A: Ubisoftは通常、すべての**ディビジョン リサージェンスのバグ報告**に対して直接的なゲーム内報酬を提供していませんが、重大なエクスプロイトやゲームを停止させる問題の報告は、コミュニティ全体に貢献します。場合によっては、主要なバグが修正された後、開発者がすべてのプレイヤーに「補償パッケージ」を提供することがあります。

**Q: バグが修正されるまでどのくらい時間がかかりますか？**

A: 修正にかかる時間は重大度によって異なります。致命的な「ブロッカー」バグは通常、数日以内にホットフィックスで対処されますが、軽微なコスメティックやUIの問題は、2026年を通じてより大規模なシーズンアップデートにまとめられる場合があります。

**Q: バグ報告ツールでプレイヤーを報告できますか？**

A: いいえ、バグ報告ツールは技術的な問題専用です。チート行為、ハラスメント、または有害な行動を報告するには、ソーシャルメニューまたはマッチ終了画面にある「プレイヤーを報告」機能を使用してください。
