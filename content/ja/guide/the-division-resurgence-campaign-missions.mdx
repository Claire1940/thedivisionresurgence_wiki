『ディビジョン リサージェンス』は、緊張感あふれるタクティカル・カバーシューター体験をモバイルデバイスで提供し、奪還すべき広大なオープンワールドのニューヨークを舞台にしています。物語に深く入り込みたいプレイヤーにとって、**ディビジョン リサージェンスのキャンペーンミッション**を探索することは、レベルアップや必須装備のアンロックに向けた主要な道筋となります。ミッドタウンの混乱に巻き込まれた第一波（ファーストウェーブ）のエージェントとして、あなたの旅は「ドルインフル」発生直後の状況を切り抜けながら、新旧両方の脅威に立ち向かうことになります。**ディビジョン リサージェンスのキャンペーンミッション**をマスターすることは、単なる戦闘だけではありません。JTF、市民の居住地、そして社会の解体を企むローグ要素の間の変化する勢力図を理解することも重要です。

## ディビジョン リサージェンス キャンペーンミッションの主要内訳

キャンペーンの初期段階は、ゲームのメカニクスとマンハッタンにおける中心的な対立を紹介する、非常に重要な導入部となります。これまでの作品とは異なり、『リサージェンス』では「スプリングブロッサム居住地」を通じて、市民の苦闘をより個人的な視点で描いています。

### スプリングブロッサムの護衛
最初の大きな任務は、スプリングブロッサム・レストランからアンジェラ・チャンとその甥のケビンを救出することです。このミッションでは「護衛」メカニクスが登場し、プレイヤーは局地的な小競り合いから防衛しつつ、市民のために進路を確保しなければなりません。JTFはミッドタウンのいくつかの検問所の制御を失っており、レストランを出てすぐに敵の抵抗に遭遇することになります。

| ミッションフェーズ | 主要目標 | 主な脅威 |
| :--- | :--- | :--- |
| **レストランからの救出** | アンジェラ・チャンと合流し、周囲の安全を確保する。 | ライカーズの初期ウェーブ |
| **進路確保** | ミッドタウン検問所の銃手を無力化する。 | 狙撃兵と側面攻撃部隊 |
| **橋の検問所** | JTFの回収地点であるマンハッタン橋に到達する。 | 強力な増援 |

### トリニティ教会の野戦病院
最初の護衛に続き、物語はドック・オニールが仮設の野戦病院を運営しているトリニティ教会へと移ります。このミッションでは、敵が.50口径機関銃やRPGなどの重火器を使用し始め、脅威レベルがエスカレートすることを強調しています。プレイヤーは、敵のリーダーであるジャックス率いる総攻撃から病院を守らなければなりません。

> **💡 ヒント：** 教会内部を防衛する際は、上部バルコニーにいるRPG射手を優先的に倒しましょう。彼らの範囲ダメージは、瞬時にアーマーを削り、人質の命を危険にさらします。

## リバティ島での決戦

プロローグのクライマックスはリバティ島で展開されます。ジャックスがドック・オニールと数名の看護師を人質に取った後、ボートでこの象徴的なランドマークまで追跡しなければなりません。このミッションは、爆弾解除を含む複雑な目的ベースの戦闘が導入されるため、**ディビジョン リサージェンスのキャンペーンミッション**を進める上で非常に重要です。

ジャックスはフェリーとその周辺エリアに爆発物を仕掛けます。パートナーが周囲をカバーしている間、プレイヤーはEOD（爆発物処理）のプロンプトに従って操作し、人質を救出する必要があります。ミッションは、高い機動力と重厚なカバーを利用するジャックスとのボス戦で締めくくられます。このアークを正常に完了すると、物語は大きく跳躍し、エージェントは1ヶ月以上の昏睡状態に陥ります。その後、「第一波」のベテランとして再び世界に足を踏み入れることになります。

## 勢力と戦闘インテル

**ディビジョン リサージェンスのキャンペーンミッション**をさらに進めていくと、敵の種類が増えていきます。代行指揮官フェイ・ラウの監視下にある医療棟で目を覚ます頃には、マンハッタンの情景は一変しています。

| 勢力 | 戦闘スタイル | 主なリーダー |
| :--- | :--- | :--- |
| **フリーメン** | 回収したハイテク機器、積極的な側面攻撃。 | 不明 |
| **ライカーズ** | 高い攻撃性、低い命中率、群れによる戦術。 | ララエ・バレット（撃破済み） |
| **クリーナーズ** | 火炎放射器とエリア拒否攻撃。 | ジョー・フェロ（撃破済み） |
| **ラスト・マン・バタリオン** | 専門的なPMC戦術、タレットの使用、回復。 | チャールズ・ブリス（撃破済み） |

「フリーメン」は2026年の物語で新たに追加された勢力で、しばしば市民の備蓄品を狙っています。彼らは技術的な独創性で知られており、改造された産業機器を使用して急造のタレットやシールドを作り出すことがよくあります。

## 居住地管理：スプリングブロッサム

**ディビジョン リサージェンスのキャンペーンミッション**の核となる要素は、スプリングブロッサム居住地の発展です。ミッドタウンの安全なエリアに位置するこのハブは、クラフト、ソーシャルインタラクション、ミッションブリーフィングの拠点となります。

*   **アンジェラ・チャン：** 人道的な側面に焦点を当て、過度な武装を見せずに「希望の光」となることを提唱しています。
*   **ケビン：** セキュリティ担当で、自動防衛システム、サーチライト、対人地雷の設置を推進しています。
*   **ドック・オニール：** クリニックを運営し、手つかずの物資の隠し場所を回収するための医療テーマのサイドクエストを提供します。

**ディビジョン リサージェンスのキャンペーンミッション**の報酬を最大化するには、これらの視点のバランスを取る必要があります。電力網や浄水システムなどの居住地インフラをアップグレードすることで、高ティアのベンダーやより優れた装備の設計図が解放されます。

## 高難易度ミッションの戦術

キャンペーンの後半ステージに挑む際、標準的なカバー間の移動だけでは不十分な場合があります。**ディビジョン リサージェンスのキャンペーンミッション**の難易度はワールドティアに応じてスケールするため、スキルと装備のより高度な相乗効果が求められます。

1.  **属性効果の連携：** 焼夷グレネードやEMPグレネードを使用して、フリーメンの技術機器を無力化しましょう。
2.  **メディックを優先：** LMBの残党が登場するミッションでは、敵の蘇生を防ぐために常にコンバットメディックを最初に排除してください。
3.  **環境の活用：** 多くのミッションエリアには爆発するドラム缶や吊り下げられた荷物があり、これらを敵の上に落とすことで弾薬を節約できます。
4.  **シグネチャーウェポンの強化：** クリントン拠点の強襲のようなボスが多いミッションに入る前に、専用キットがレベルアップしていることを確認してください。

## FAQ

**Q: ディビジョン リサージェンスのキャンペーンミッションはソロでプレイできますか？**

A: はい、ストーリーキャンペーン全体がソロでプレイできるように設計されています。ただし、難易度はグループの人数に合わせて調整されるため、友人を連れて行くことで一部の「ホード」形式の防衛目標が管理しやすくなります。

**Q: 第二波（セカンドウェーブ）のミッションを解放するにはどうすればよいですか？**

A: 第二波のストーリーラインは、リバティ島のアークとそれに続く「昏睡」の転換点を完了した後に利用可能になります。作戦基地でフェイ・ラウからブリーフィングを受けることで、これらの新しい章を開始できます。

**Q: ミッション中に人質が死ぬとどうなりますか？**

A: ほとんどのキャンペーンミッションには、重要なNPCに対する「失敗条件」があります。ドック・オニールのような人質が殺された場合、通常は直前のチェックポイントからリスタートとなります。これらのシークエンスでは、非戦闘員から敵の注意をそらすようにしてください。

**Q: 初期キャンペーンに取り逃しの報酬はありますか？**

A: ほとんどの報酬はミッション完了に紐付いており、「取り逃す」ことはありません。ただし、ミッション中に脇道にある部屋を探索することで、2026年のニューヨークの状況に関する追加のXPや伝承の背景を提供する「エコー」や音声ログを見つけることができます。
