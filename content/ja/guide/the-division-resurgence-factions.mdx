## フリーマン：2026年の新たな脅威

本作のロア（設定）における最も重要な追加要素の一つが、フリーマンの導入です。明晰ながらも幻滅したアーサー・キング率いるこの勢力は、政府権力に対する急進的な不信感から生まれた、これまでとは異なる種類の脅威を象徴しています。アーサー・キングはかつてシリコンバレーで賞賛された未来学者であり発明家でしたが、ドルインフルによる社会の崩壊後、悲劇的な出来事によって暗く復讐心に満ちた考えに陥りました。

フリーマンは、連邦政府がパンデミックの直接的な責任を負っており、ディビジョンは単なる抑圧の道具であると信じています。混乱の最中に妻と息子を失ったことでキングは「壊れ」、自身の技術的な天才性を戦争へと転用しました。フリーマンは単なる無秩序な暴徒ではありません。高度なテクノロジーとゲリラ戦術を駆使して、街に残されたインフラを解体しようとする、目的を持った武装集団です。

| 特徴 | 詳細 |
| :--- | :--- |
| **リーダー** | **アーサー・キング**（元シリコンバレーの発明家） |
| **主な動機** | 政府への復讐と完全な自治 |
| **戦闘スタイル** | ガジェット、ドローン、組織的な待ち伏せの戦術的利用 |
| **主な弱点** | 技術的ハードウェアへの過度な依存と特定のリーダーノード |

## クリーナーズ：炎による浄化

クリーナーズは、街を「浄化」するという狂信的な献身により、依然として最も恐ろしい**ディビジョン リサージェンスの勢力**の一つです。主に元清掃作業員やブルーカラーの労働者で構成されており、ウイルスを止める唯一の方法は、感染の可能性があるものすべて、そして全員を焼き払うことだと信じています。

戦闘において、クリーナーズは攻撃的で、火炎放射器、焼夷手榴弾、頑丈な保護具を使用します。彼らの「ヘビー」は特に危険で、厚い装甲に身を包み、エージェントを数秒でカバーから炙り出す巨大な火炎放射器を操ります。しかし、彼らが加圧タンクに依存していることは、熟練した射手にとって大きな戦術的利点となります。

> **警告：** クリーナーズと交戦する際は、常に燃料タンクを優先してください。正確な一撃は壊滅的な爆発を引き起こし、ターゲットを無力化すると同時に周囲の敵にもダメージを与えます。

### クリーナーズのエネミーアーキタイプ

*   **スイーパー：** 火炎放射器と盾を装備した基本歩兵。
*   **シールド：** 前進してプレイヤーをカバーから追い出すタンクユニット。
*   **インシネレーター：** 膨大な体力と遠距離火炎攻撃能力を持つ重量級ユニット。

## ライカーズ：混沌と信念

崩壊初期に悪名高いライカーズ島刑務所から脱獄したライカーズは、強欲と「弱肉強食」の精神によって定義される勢力です。彼らはニューヨークの崩壊を「街の新たな王」になるチャンスと捉えています。歪んだ目的意識を持つクリーナーズとは異なり、ライカーズは権力、富、そして自分たちを投獄したシステムへの復讐心によって動いています。

ライカーズは混沌とした高圧的な戦術を採用します。圧倒的な数でエージェントの側面を突き、盗んだ警察の装備を利用して防御を固めることがよくあります。この勢力に対処するには、包囲されないように常に周囲の状況を把握しておく必要があります。

| ユニットタイプ | 装備武器 | 戦術的行動 |
| :--- | :--- | :--- |
| **ライカー・ラッシャー** | ショットガン / 近接武器 | カバーを壊すためにプレイヤーに直進してくる |
| **ライカー・ガード** | LMG / 重装甲 | 側面攻撃ユニットのために制圧射撃を行う |
| **ライカー・スナイパー** | マークスマンライフル | 遠距離を保ち、高所を利用する |

## 戦略自衛局 (SHD)

厳密にはプレイヤーの組織ですが、ディビジョンはゲームのエコシステム内で一つの勢力として機能しています。私たちの使命は「残されたものを守り」、完全に壊滅する寸前の街に秩序を取り戻すことです。エージェントとして、ISAC（インテリジェント・システム・アナリティック・コンピューター）や様々な戦闘スキルを含む、2026年時点で利用可能な最も高度なテクノロジーにアクセスできます。

ディビジョンの強みはその汎用性にあります。敵対する**ディビジョン リサージェンスの勢力**が専門化しているのに対し、エージェントは特定の脅威に対抗するためにロードアウトを適応させることができます。ライカーズを炙り出すために追尾マインを使ったり、フリーマンの電子機器を妨害するためにケミランチャーを使ったりと、SHDは都市の戦場における究極のワイルドカードです。

## 2026年の戦闘戦略

異なる勢力と交戦するには、考え方を切り替える必要があります。フリーマンの集団に対して、クリーナーズの分隊と同じようにアプローチすることはできません。効率を最大化するために、以下の勢力別プロトコルに従ってください。

1.  **フリーマン対策：** 彼らのテクノロジーを妨害することに集中してください。EMPパルスやグレネードを使用して、ドローンやタレットを無効化します。エンジニアを早めに標的にすることで、さらなるガジェットの展開を防げます。
2.  **クリーナーズ対策：** 距離を保ってください。彼らの火炎放射器は射程が限られていますが、至近距離では致命的です。背中や腰にある明るい赤色の酸素・燃料タンクを狙いましょう。
3.  **ライカーズ対策：** 側面に注意してください。ライカーズはリーダーが制圧射撃を行っている間にラッシャーを送り込むことを好みます。ライオットフォームやスティンガーハイブなどのクラウドコントロールスキルを使用して、移動ルートを封鎖しましょう。

装備やミッションの場所に関する詳細な情報については、[公式 The Division Resurgence ウェブサイト](https://www.ubisoft.com/en-us/game/the-division/resurgence)を訪問し、最新のシーズンイベントや勢力の拡張情報を確認してください。

## 勢力別のドロップアイテムと報酬

各**ディビジョン リサージェンスの勢力**は、ビルドの調整に役立つ特定のタイプのアイテムをドロップします。最適化されたギアセットを探しているエンドゲームプレイヤーにとって、特定の勢力を周回することは一般的な戦略です。

| 勢力 | 一般的なドロップタイプ | レアアイテムの可能性 |
| :--- | :--- | :--- |
| **フリーマン** | 電子機器 ＆ スキルMOD | ハイテクなエキゾチック部品 |
| **クリーナーズ** | 攻撃用装備 ＆ 武器パーツ | 焼夷耐性ギア |
| **ライカーズ** | 防御用装備 ＆ アーマープレート | 高ダメージのショットガン ＆ SMG |

> **プロのヒント：** 作戦基地にあるデイリープロジェクトボードを確認してください。多くの場合、これらのミッションでは特定の勢力の無力化が課せられ、ボーナスXPや特殊なキャッシュが提供されます。

## 結論

2026年のニューヨークの景観は、対立するイデオロギーと危険な敵が入り混じる複雑な網の目のようなものです。**ディビジョン リサージェンスの勢力**の動機と戦術を理解することで、今後の課題に備えることができます。アーサー・キング率いるフリーマンの技術的脅威から、クリーナーズの焦土作戦まで、あらゆる遭遇はエージェントとしてのスキルと適応力のテストとなります。警戒を怠らず、背後に注意し、そして使命を忘れないでください。残されたものを守るのです。

## FAQ

**Q: 『ディビジョン リサージェンス』のフリーマンとは誰ですか？**

A: フリーマンは、元シリコンバレーの発明家アーサー・キング率いる新しい敵勢力です。彼らは反政府の急進派であり、社会崩壊の責任がディビジョンにあると考え、高度なテクノロジーやガジェットを駆使して戦います。

**Q: クリーナーズのヘビーを倒すにはどうすればいいですか？**

A: クリーナーズのヘビーを倒すには、背中や足にある赤い加圧タンクに射撃を集中させてください。これらのタンクが破裂すると爆発し、ヘビーに大ダメージを与え、即座に倒せる可能性があります。

**Q: ゲーム内に味方の勢力はいますか？**

A: はい、プレイヤーはしばしばJTF（合同タスクフォース）や様々な市民の居住地と協力します。これらのグループは、オープンワールド全体でサポート、ベンダー、クエストの目的を提供してくれます。

**Q: 『ディビジョン リサージェンス』の勢力の中で、戦うのが最も難しいのはどれですか？**

A: 難易度はプレイスタイルによります。フリーマンはドローンやタレットを使用するため手強いとされており、ライカーズは攻撃的な側面攻撃と数の多さから苦戦することがあります。
