## 金庫の仕組みと手がかりを理解する

『ディビジョン リサージェンス』において、金庫は「興味深いオブジェクト」として分類されます。金庫の近くにいると、ISACシステムが通常「興味深いオブジェクトを検知しました」とアナウンスします。しかし、金庫にインタラクトすることは戦いの半分に過ぎません。各金庫には4桁または3桁の暗証番号が必要です。

これらの組み合わせはランダムではなく、ゲームの世界に組み込まれています。以下の場所で見つけることができます：
*   **黒板：** 拠点や教室でよく見られます。
*   **ポスターや看板：** 店の窓や建物のファサードにある「Established（設立年）」の日付を確認してください。
*   **収集アイテム：** 特に「暗号の手がかり」メニューでは、暗号の手がかり07のようなアイテムを追跡でき、これらは謎解きや日付、出来事への直接的な言及を提供します。

> **注意：** 一部の金庫はエリートの敵によって守られているか、部屋にアクセスできるようになる前に特定の「ワールドイベント」をクリアする必要があります。コードを入力する前に、必ず周囲の安全を確保してください。

## 詳細解説：暗号の手がかり07 (The Division Resurgence Code Clue 07)

多くの金庫が単純な視覚的ヒントを使用している一方で、**the division resurgence code clue 07**を含む「暗号の手がかり」シリーズは、もう少し調査が必要です。これらは多くの場合、マンハッタンを安定させようとするSHDの取り組みという、より大きな物語に関連しています。

暗号の手がかりを拾うと、それは「コレクション」タブに保存されます。手がかり07については、プレイヤーは金融街の周辺や、JTFの残党が関与する特定のサイドミッション中に見つけることが多いです。この手がかりは通常、重要な日付や、伝承エントリに見られる特定の部屋番号を指しています。

### 暗号パズルの解き方
1.  **インベントリメニューを開く：** 「収集アイテム」セクションに移動します。
2.  **暗号の手がかりを選択：** 手がかり07またはそれに相当するもの（「誕生日プレゼント」の手がかりなど）を探します。
3.  **テキストを分析する：** 手がかりに「5月11日火曜日」と書かれている場合、コードは**1105**または**0511**である可能性が高いです。
4.  **環境内の日付を確認する：** 歴史的な建物にある金庫の場合、「Established [西暦]」の看板を探してください。もし年が1883年なら、それが組み合わせになります。

## 作戦基地の隠し金庫

作戦基地 (BoO) は単なるソーシャルハブではありません。初心者が見落としがちな金庫がいくつか含まれています。これらはダークゾーンに足を踏み入れる前に、ビルドのスタートダッシュを切るのに最適です。

### 宿舎の金庫
ファストトラベル後にエージェントがスポーンする居住区にあります。就寝スペースに入る前に掲示板を探してください。小さな紙切れにコード**1350**が書かれています。この金庫は、序盤の武器MODの設計図をドロップすることが多いため非常に重要です。

### マディソン通りループ
基地のすぐ外に出て、マディソン野戦病院（第1作目へのオマージュ）の方へ左に曲がると、一連の店舗があります。
*   **セロテンティアの金庫：** これは大型の床置き金庫です。これを開けるには、まずテントの外の地面にある収集アイテム「誕生日プレゼント」を拾う必要があります。手がかりには11月5日と言及されているため、コードは**1105**になります。
*   **スーパーマーケット：** オークロック・ストアの向かいにあります。冷凍食品コーナーの上の大きな赤い看板にコード**1987**が見えます。

> **💡 ヒント：** スペシャリゼーションツリーの「スカベンジング」スキルを常にアップグレードしておくと、これらのワールド金庫からドロップする戦利品の品質が向上します。

## 2026年の探索戦略

2026年にゲームが進化するにつれ、新しい地区が追加され、それぞれに独自の金庫と暗号の手がかりが用意されています。常に一歩先を行くために、以下の探索チェックリストに従ってください：

1.  **セーフハウスへのファストトラベル：** すべてのセーフハウス（オータムズ・ホープ、ダンテズ・ランなど）には、通常、入り口から100メートル以内に金庫が1つ隠されています。
2.  **黒板を探す：** オフィスや教室のように見える部屋では、黒板をチェックしてください。開発者はしばしば「白いチョーク」のテクスチャを使って、目に見える場所にコードを隠しています。
3.  **すべての収集アイテムにインタラクトする：** コンプリートを目指していない場合でも、暗号の手がかり07のような収集アイテムは、エキゾチック部品を含む価値の高い「マスター金庫」を開ける唯一の方法です。
4.  **ヘッドフォンを使用する：** ISACの「興味深いオブジェクト」のオーディオキューには方向性があります。ピンという音が聞こえたら、金庫は近くのデスクの後ろや小さなクローゼットの中にある可能性が高いです。

すべての金庫が同じ品質のアイテムを提供するわけではありません。報酬はワールドティアや、あなたがいるゾーンの難易度に応じてスケールします。

| 金庫のタイプ | 一般的な内容物 | 最適な用途 |
|---|---|---|
| **壁掛け金庫** | クレジット、消耗品 | ミッション中の補給 |
| **床置き金庫** | アーマー、武器MOD | ビルドの微調整 |

## FAQ

**Q: 暗号の手がかり07はどこで見つかりますか？**

A: この手がかりは「暗号の手がかり」収集セットの一部です。通常、ゲームの後半の地区、特にオフィスビルや放棄されたSHD中継ステーション内で見つかります。これを拾うことで、近くの高セキュリティ金庫に必要なヒントが得られます。

**Q: 金庫のコードは毎週変わりますか？**

A: いいえ、『ディビジョン リサージェンス』の金庫コードは固定されており、毎週のリセットで変わることはありません。一度コードを知れば、2026年シーズン中、あなたのアカウントのすべてのキャラクターで使用できます。

**Q: コードを知っているのに金庫にインタラクトできないのはなぜですか？**

A: 一部の金庫はミッションの進行状況によってロックされています。特定のレベルに達していないか、その地域の「コントロールポイント」を制圧していない場合、インタラクトのプロンプトが表示されないことがあります。まずマップ上でそのエリアが「緑色（安全）」になっていることを確認してください。

**Q: これらの金庫にエキゾチックは入っていますか？**

A: 通常のワールド金庫でエキゾチックが出ることは非常に稀ですが、長い暗号の手がかりに関連する「マスター金庫」には、トップティアの武器を作成するために使用される「エキゾチック部品」がドロップするわずかな確率があります。
