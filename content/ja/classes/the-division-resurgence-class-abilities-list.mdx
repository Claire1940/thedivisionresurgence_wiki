## ブルワーク：究極の最前線ディフェンダー

ブルワークは、高い生存率を誇る分隊の要です。突破作戦と戦闘耐久の専門家として、このクラスは敵の激しい防御を突き破りながらダメージを吸収するように設計されています。「タンク」プレイスタイルを好むなら、ブルワークは自分自身とチームメイトの両方に比類のない保護を提供します。

ブルワークの最大の特徴は、防御姿勢を維持しながら攻撃を続けられる能力です。激しい砲火の下で撤退が必要になる可能性のある他のクラスとは異なり、ブルワークはシールドを使用して陣地を維持します。

| アビリティタイプ | 名称 | 説明 |
| :--- | :--- | :--- |
| **シグネチャー** | **ブリーチコンボ** | 拡張されたフランクシールドを展開し、至近距離で最大の殺傷力を発揮する**612オートマチックショットガン**を装備する。 |
| **コンバットスキル** | **フランクシールド** | エージェントがいかなる武器も発射できるようにしながら、上半身を保護する。 |
| **コンバットスキル** | **ショックウェーブスパイク** | ダッシュ中に叩きつけたり、遠くの敵を威嚇するために投げたりできる設置型スパイク。 |

> **💡 ヒント：** ショックウェーブスパイクMK3の回収メカニズムを活用しましょう。手動で拾い上げることでクールダウンの一部が返還され、より頻繁なクラウドコントロールが可能になります。

## デモリショニスト：爆発的なエリア拒否

高い衝撃力を持つ範囲攻撃（AoE）ダメージを好むエージェントにとって、デモリショニストは最高の選択肢です。このクラスは、手榴弾、マイン、重火器を使用して戦闘空間を支配することに長けています。特に敵をカバーから追い出したり、密集した敵集団を殲滅したりするのに効果的です。

デモリショニストの「フォーカス」パスでは、爆発物の純粋なダメージを増加させるか、炎上や出血などのガジェットによるステータス効果を向上させるかを選択できます。

| スキル | 主な機能 | 戦術的用途 |
| :--- | :--- | :--- |
| **グレネードランチャー** | シグネチャーウェポン | 複数の高爆発弾を連続して発射する。 |
| **アーティラリータレット** | コンバットスキル | 離れた場所から指定されたターゲットに迫撃砲スタイルの弾丸を放つ。 |
| **爆発シーカーマイン** | コンバットスキル | 敵を追跡し、衝撃時に爆発して広範囲に大ダメージを与える。 |

**ディビジョン リサージェンスのクラスアビリティ一覧**を見ると、デモリショニストはミッションでの「雑魚敵」の一掃に最適なオプションとして際立っています。アーティラリータレットは『ディビジョン2』と同様に手動での照準が必要ですが、高所に設置することで壊滅的な成果をもたらします。

## テックオペレーター：ロボティクスの達人

テックオペレーターは、SHDドローンのメンテナンスと展開に焦点を当てています。このスペシャライゼーションは、自動化されたシステムを使用して敵を翻弄したり、戦術的なバフを提供したりする「ペット」プレイスタイルを楽しむプレイヤーに最適です。敵を混乱させるという点において、テックオペレーターの有用性は他に類を見ません。

このクラスの核となるのは、ダメージを与えるだけでなく既存のガジェットを強化するユニークなシグネチャーアビリティ、**SHDブリック**です。

*   **SHDブリック（シグネチャー）：** 展開されたストライカードローンをオーバーロードさせる波動を放ち、一時的にダメージ出力を大幅に増加させます。ドローンが稼働していない場合は、自動的に1機展開します。
*   **ストライカードローン（コンバットスキル）：** エージェントに追従し、ターゲットを自動的に攻撃する自律型ドローン。

**ディビジョン リサージェンスのクラスアビリティ一覧**に見られる奥深さは、コンソール版のそれにも匹敵します。特にテックオペレーターの「フォーカス」タレントでは、ドローンの耐久性に特化するか、クールダウンの短縮を優先して常にロボットの相棒をそばに置くかを選択できます。

## ヴァンガード：戦術的認識とバフ

ヴァンガードは「槍の先端」として機能し、チームに不可欠な戦闘データとダメージバフを提供します。このクラスは、初代『ディビジョン』の象徴的な**タクティカルリンク**を復活させました。これはPvPシナリオでのバーストダメージにおいて最も強力なアビリティの一つと見なされています。

| アビリティ | 効果 |
| :--- | :--- |
| **タクティカルリンク** | エージェントと近くの味方に大幅なダメージ増加と「エイムボットのような」精度を与えるシグネチャーバフ。 |
| **スマートカバー** | カバーを強化し、ダメージ軽減と命中時に弾丸を補充するチャンスを与える。 |
| **スキャナーパルス** | 周囲をスキャンして敵をハイライトし、敵が受けるクリティカルヒットダメージを増加させる。 |

ヴァンガードは対人戦において不可欠です。ダークゾーンでは、エリアをパルスして潜伏しているローグや敵NPCを検知できる能力が、部隊に先制権を与えます。**ディビジョン リサージェンスのクラスアビリティ一覧**に基づいてビルドを更新し、プレッシャーの高いゾーンで競争力を維持しましょう。

## フィールドメディック：支援と腐食ユーティリティ

フィールドメディックは単なるヒーラー以上の存在です。分隊に最高の持続力を提供する一方で、敵のアーマーや体力を徐々に削る攻撃的な「腐食」アビリティも備えています。この二重の役割により、フィールドメディックはどのようなチーム構成でも重宝される汎用性の高い選択肢となります。

**オキシダイジング・スウォーム**は際立ったシグネチャーアビリティです。ハイヴに似た機能を持ちますが、強力な腐食性物質を充填したマイクロドローンを放出します。これは敵をカバーから強制的に追い出したり、高アーマーの「チュンガス」ユニットを溶かしたりするのに最適です。

> **警告：** フィールドメディックは攻撃的に立ち回ることもできますが、常にチームメイトの体力バーに注意を払ってください。英雄難易度では、あなたのサポートステーションだけが全滅を防ぐ唯一の手段となることがよくあります。

### メディックアビリティ比較

| スキル名 | 役割 | メカニズム |
| :--- | :--- | :--- |
| **オキシダイジング・スウォーム** | 攻撃/ユーティリティ | 敵のアーマーを侵食する腐食性マイクロドローンを放出する。 |
| **オキシダイザートラップ** | クラウドコントロール | 範囲内に敵が入ると発動する腐食性トラップを散布する。 |
| **サポートステーション** | 回復 | 緑色の円の範囲内にいるすべての味方を回復する設置型デバイスを展開する。 |

## フォーカスタレントによるビルドの最大化

ディビジョン リサージェンスの各クラスには、2つの「フォーカス」ツリーがあります。これらは役割をさらに洗練させるための専門化されたタレントパスです。例えば、ブルワークには「ブリーチング」（攻撃的なシールドプレイ）に特化したフォーカスと、純粋な「プロテクション」（防御的なチームバフ）に特化したフォーカスがあります。

1.  **ミッションを分析する：** 装甲の厚いボスに直面する場合は、アーマー削りのためにデモリショニストやフィールドメディックに焦点を当てましょう。
2.  **チームメイトと相乗効果を高める：** ヴァンガードのパルスとテックオペレーターのオーバーロードされたドローンを組み合わせれば、数秒で敵の波を一掃できます。
3.  **クールダウンを管理する：** ブルワークやデモリショニストなど、多くのクラスには展開したガジェットと相互作用することでスキルチャージを回復する手段が用意されています。

## FAQ

**Q: ディビジョン リサージェンスでは、いつでもクラスを切り替えられますか？**

A: はい、解放済みのスペシャライゼーション間であれば切り替え可能です。ただし、新しいクラスの要件にステータスを合わせるために、装備や「フォーカス」タレントを調整する必要がある場合があります。

**Q: 最新のディビジョン リサージェンスのクラスアビリティ一覧はどこで確認できますか？**

A: このガイドは2026年版に更新されています。ただし、シーズンパッチで導入される最新のバランス調整やタレントの変更については、常にゲーム内のスペシャライゼーションメニューを確認してください。

**Q: ソロプレイヤーに最適なクラスはどれですか？**

A: 一般的にブルワークとテックオペレーターがソロプレイに最適とされています。ブルワークはヒーラーなしでも生き残るための生存力を提供し、テックオペレーターのドローンはヘイトを稼ぐ第2のチームメイトとして機能します。

**Q: シグネチャーアビリティはどうすれば解放できますか？**

A: シグネチャーアビリティはスペシャライゼーションに紐付いています。クラスを選択し、初期のチュートリアルと序盤のレベルを進めると、シグネチャーウェポンとそれに対応するアビリティが戦闘で使用可能になります。
