## コントローラーの接続方法

**ディビジョン リサージェンスのゲームパッド対応**の設定は簡単なプロセスですが、モバイルOSによって若干異なります。重要なミッション中に接続が切れないよう、ペアリングを試みる前にコントローラーが十分に充電されていることを確認してください。

### Androidでの設定
1. デバイスの**設定**を開き、**Bluetooth**に移動します。
2. コントローラーをペアリングモードにします（例：DualSenseの場合はShareボタンとPSボタンを長押し）。
3. 利用可能なデバイスの一覧からコントローラーを選択します。
4. 『ディビジョン リサージェンス』を起動します。ゲーム開始時にコントローラーのチュートリアルが表示されます。

### iOSでの設定
1. **設定** > **Bluetooth**に移動し、オンになっていることを確認します。
2. ゲームパッドのペアリングモードを起動します。
3. 「その他のデバイス」の下にあるデバイス名をタップします。
4. 「接続済み」と表示されたら、ゲームを開きます。ゲーム内の設定メニューで接続を確認できます。

## デフォルトのボタンマッピングと操作方法

**ディビジョン リサージェンスのゲームパッド対応**のデフォルトレイアウトは、三人称視点シューティング（TPS）の業界標準に従っています。シリーズの過去作をプレイしたことがあれば、すぐに馴染めるでしょう。複雑なRPGシステムを管理するために、標準的なコントローラーのほぼすべてのボタンが使用されます。

| アクション | Xbox 入力 | PlayStation 入力 |
|---|---|---|
| **移動 / ダッシュ** | 左スティック（押し込みでダッシュ） | L3 |
| **エイム / カメラ** | 右スティック（押し込みで格闘） | R3 |
| **射撃 / 攻撃** | RT | R2 |
| **照準（ADS）** | LT | L2 |
| **カバー / 乗り越え** | A | ×ボタン |
| **リロード / インタラクト** | X | □ボタン |
| **武器切り替え** | Y | △ボタン |
| **スキル1使用** | LB | L1 |
| **スキル2使用** | RB | R1 |
| **シグネチャーアビリティ** | LB + RB | L1 + R1 |

> **⚠️ 注意:** 一部の古いBluetoothコントローラーでは、スティックがニュートラル状態でもキャラクターがわずかに動いてしまう「デッドゾーン」の問題が発生することがあります。オプションメニューの「ゲームプレイ」タブでデッドゾーンの設定を調整できます。

## スペシャリゼーションに合わせたパフォーマンスの最適化

『ディビジョン リサージェンス』の各クラスは、特定のコントローラー調整によって恩恵を受けます。デモリショニストやヴァンガードなど、多様な役割が存在するため、感度設定はプレイスタイルを反映させるべきです。

### デモリショニスト（爆発物特化）
デモリショニストは、グレネードやシグネチャーウェポンを正確に配置することに依存します。このクラスでは、グレネードの軌道を完璧に合わせるために「ADS感度」を少し低めに設定することをお勧めします。追尾マインやタレットのためにL1/R1を頻繁に使用するため、バンパーボタンの反応が良いことを確認してください。

### ヴァンガード（サポートおよびタンク）
ヴァンガードとしての主な目標は、ポジショニングと盾の管理です。**ディビジョン リサージェンスのゲームパッド対応**により、タッチ操作に比べて盾を展開した状態での移動がよりスムーズになります。側面を突こうとする敵に素早く反応できるよう、「旋回感度」を上げるのも良いでしょう。

### フィールドメディック（ヒーラー）
回復には素早いターゲットの切り替えが必要です。画面上の小さなアイコンをタップするよりも、D-Pad（方向キー）を使用してチームメイトを切り替える方が速い場合が多いです。戦闘の合間にタクティカルマップやソーシャルメニューを操作できるよう、「インターフェース感度」を十分に高く設定しておきましょう。

## タッチ操作 vs ゲームパッド：どちらが良いか？

開発者はモバイル専用プレイヤー向けにタッチインターフェースを最適化していますが、ハイレベルなプレイにおいては**ディビジョン リサージェンスのゲームパッド対応**が否定できない利点を提供します。以下は、2つの操作体系の比較です。

| カテゴリー | タッチ操作 | ゲームパッド対応 |
|---|---|---|
| **精密エイム** | 普通（エイムアシストに依存） | **高い**（微細な指のコントロールが可能） |
| **移動の滑らかさ** | 低い（画面が指で塞がる） | **高い**（専用のアナログスティック） |
| **画面の視認性** | 悪い（指が視界を遮る） | **非常に良い**（クリーンなUI） |
| **反応速度** | 速い（直接タップ） | **より速い**（物理的な触覚フィードバック） |
| **携帯性** | **最高**（追加の機材不要） | 普通（コントローラーの持ち運びが必要） |

コンフリクト（PvPモード）でランクを上げたい競争志向のプレイヤーにとって、コントローラーはほぼ必須と言えます。「クイックスコープ」や迅速なカバー間移動を行う能力は、ガラスの画面と格闘していない時の方が格段に容易になります。公式のハードウェア要件の詳細については、[Ubisoft サポートページ](https://www.ubisoft.com/ja-jp/game/the-division/resurgence)で確認できます。

## よくあるゲームパッドの問題のトラブルシューティング

**ディビジョン リサージェンスのゲームパッド対応**が正しく機能しない場合は、以下の一般的な修正方法を試してください。

1. **ボタンが反応しない:** これは、ゲームを起動した*後*にコントローラーを接続した場合によく起こります。アプリを完全に閉じ、スマホのOSでコントローラーがペアリングされていることを確認してから、ゲームを再起動してください。
2. **キーガイドの表示が正しくない:** PlayStationコントローラーを使用しているのにXboxのボタンガイドが表示される場合は、メニューの「コントローラータイプ」設定を確認してください。ベータ版などでは、デフォルトでXboxアイコンになっている場合があります。
3. **ゴースティング / 遅延:** これは通常、他のBluetoothデバイスとの干渉によって引き起こされます。近くにあるタブレットやPCのBluetoothをオフにして、信号をクリアにしてみてください。
4. **ファームウェアの更新:** DualSenseやXbox Seriesコントローラーなどの最新コントローラーは、ファームウェアの更新が必要です。コンソールまたはPCに接続して、モバイルとの互換性を最大限に高めるために最新のソフトウェアを実行していることを確認してください。

## FAQ

**Q: 『ディビジョン リサージェンス』はすべてのクラスでゲームパッドを使用できますか？**

A: はい、**ディビジョン リサージェンスのゲームパッド対応**は、デモリショニスト、ブルワーク、フィールドメディック、ヴァンガードを含むすべてのスペシャリゼーションで共通して利用可能です。ボタンマッピングは一貫していますが、クラス固有のスキルはショルダーボタン（L1/R1 または LB/RB）に割り当てられます。

**Q: ボタン配置をカスタマイズすることはできますか？**

A: いくつかのプリセット構成が用意されているほか、設定で限定的なボタンリマッピングが可能です。「カバー」と「ダッシュ」の入力を入れ替えるなど、好みに合わせて特定のボタン機能を変更できます。

**Q: コントローラー使用時にエイムアシストはありますか？**

A: はい、ゲームパッドユーザー向けに調整可能なエイムアシスト機能が用意されています。これにより、スティックとマウスの操作性の違いを補完し、ゲームに「遊ばされている」感覚を与えることなく、スムーズでやりがいのある体験を提供します。

**Q: コントローラーを使用すると、マッチメイキングのプールが分けられますか？**

A: 2026年現在、Ubisoftは待ち時間を短縮するために、ほとんどのモードでタッチ操作ユーザーとコントローラーユーザーのクロスプレいを導入しています。ただし、一部のハイレベルなランクマッチでは、公平性を保つために入力デバイスに基づいたマッチメイキングが採用される場合があります。
