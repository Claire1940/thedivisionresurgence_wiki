現代文明の崩壊は、一瞬の出来事だった。「グリーンポイズン」あるいは「ドルインフル」として知られる人為的な病原体が引き金となったのだ。『ディビジョン リサージェンス』のストーリーにおいて、プレイヤーはアウトブレイク初期の雪に覆われた混沌としたニューヨークの街へと引き戻される。ストラテジック・ホームランド・ディビジョン（SHD）の一員として、あなたはあらゆる手段が尽きた時に秩序を回復するために招集された潜伏エージェント「第一波（ファーストウェーブ）」の一部となる。シリーズ初のモバイル作品となる本作は、第1作目とその続編の間の空白を埋めつつ、新たなキャラクターや危険な脅威を導入し、タイムラインに対する独自の視点を提供する。

『ディビジョン リサージェンス』のストーリーを理解するには、合同タスクフォース（JTF）がマンハッタンの統制を完全に失う前に起こった特定の出来事を深く掘り下げる必要がある。ベテランプレイヤーには馴染みのある設定かもしれないが、本作では危機の絶頂期における市民の保護や重要資産の確保を巡る、より詳細で絶望的な戦いが描かれている。

## アウトブレイクのタイムライン：ブラックフライデーから崩壊まで

ディビジョンの世界観における物語は、急速な衰退の連鎖によって定義される。メインシリーズがその後の余波に焦点を当てているのに対し、『リサージェンス』は機能していた都市が隔離された戦場へと変貌していく過程を強調している。以下の表は、2026年のゲーム設定に至るまでの重要な出来事をまとめたものである。

| 時期 | 出来事 | ニューヨークへの影響 |
| :--- | :--- | :--- |
| **11月下旬** | グリーンポイズンの拡散 | ブラックフライデーのセール中にマンハッタンが感染。 |
| **12月上旬** | 指令51の発令 | SHDエージェントが召集され、都市機能が崩壊。 |
| **12月中旬** | 第一波の展開 | エージェントがトリニティ教会などの重要拠点の確保を試みる。 |
| **12月下旬** | JTFの撤退 | 軍隊がミッドタウンの検問所の制御を失う。 |
| **1月 - 5月** | 全米への拡大 | ウイルスがワシントンD.C.、ボストン、シルバークリークなどの地方に到達。 |

タイムラインが示す通り、『ディビジョン リサージェンス』のストーリーは極めて不安定な時期を舞台にしている。ディビジョンが拠点を確立した後の作品とは異なり、ここではエージェントは限られたリソースと情報の中で、目の前の惨劇に即座に対応しなければならない。

## 『ディビジョン リサージェンス』のストーリーとロアの分析

ゲームは、文明の崩壊を詳細に描いた印象的なシネマティックの直後から始まる。プレイヤーは第一波エージェントとして招集され、地方政府の残存勢力を支援する任務に就く。主要な連絡先の一人は、ロウアー・マンハッタンのセクションチーフ代行である**アストリッド・マヤ**だ。彼女は、悪党たちが野放しで活動する街をナビゲートするために必要な戦術的監視を提供する。

序盤のミッションは、リスクの高い救出作戦に焦点を当てている。最初の大きな目的の一つは、エージェント・ケリーの居場所を突き止め、スプリング・ブロッサム・レストランでVIPの**アンジェラ・チャン**を救出することだ。このミッションは、ディビジョンの冷徹な戦術的目的と、仲間を置き去りにすることを拒む生存者たちの人間的要素との間の摩擦という、物語の繰り返されるテーマを浮き彫りにしている。

### 物語の主要キャラクター

| キャラクター | 役割 | 重要性 |
| :--- | :--- | :--- |
| **アストリッド・マヤ** | SHDセクションチーフ | ロウアー・マンハッタンでのエージェント活動を統括。 |
| **エージェント・ケリー** | フィールドエージェント | 初期救出作戦における主要な味方。 |
| **アンジェラ・チャン** | VIP / 科学者 | 都市の復興に深く関わる重要人物。 |
| **Dr. オニール** | 医療スタッフ | トリニティ教会野戦病院の主任医師。 |
| **ジャックス** | 敵対者 | リバティ島を狙う敵対勢力のリーダー。 |

> **警告：** この初期段階では、敵対勢力はJTFよりも重武装である場合が多い。重火器のキャッシュが敵の手に渡るのを防ぐため、常に奪還を優先すること。

## トリニティ教会とリバティ島を巡る戦い

『ディビジョン リサージェンス』のストーリーにおける極めて重要な瞬間は、トリニティ教会の野戦病院が激しい攻撃を受けた時に訪れる。ジャックスという冷酷な人物に率いられたレイダーズが陣地を制圧し、Dr. オニールを含む数名を人質に取る。このシークエンスは、軍用の支援兵器が敵対勢力に奪われるなど、JTFの権威が崩壊していく様子を如実に示している。

ジャックスの追跡は、証券取引所を経て、最終的に**リバティ島**へと続く。この場所は単なるランドマークではなく、戦略的な物資供給ハブとして機能している。ジャックスは物資を積んだフェリーで脱出し、追跡を逃れるために島を爆破しようと計画する。エージェントが仕掛けられた爆弾を解除して看護師やDr. オニールを救い、ドックでジャックスと対峙する場面で物語の緊張感は最高潮に達する。

## 勢力と新たな脅威

中央政府の影響力が弱まるにつれ、いくつかの勢力が権力の空白を埋めるために台頭する。マンハッタン地域では「フリーメン」が注目すべき脅威となっているが、地方セクターでは「ヴァルチャーズ」や「リーパーズ」といったグループも現れ始めている。『ディビジョン リサージェンス』のストーリーは、これらのグループがいかにして単なる略奪者から組織化された準軍事組織へと変貌を遂げたのかという背景を提供している。

1.  **レイダーズ：** 無秩序だが数は多く、医療スタッフの誘拐や物資の略奪に焦点を当てている。
2.  **フリーメン：** 街を悩ませる新たな脅威であり、縄張りを巡って第一波エージェントとしばしば衝突する。
3.  **JTF（合同タスクフォース）：** 名目的には味方だが、指揮系統の崩壊や相次ぐ脱走により、2026年時点では信頼に欠けるパートナーとなっている。

これらの勢力に関する公式の詳細については、[Ubisoft リサージェンス公式サイト](https://www.ubisoft.com/en-us/game/the-division/resurgence)を参照。

## ロアの繋がり：ニューヨークからシルバークリークへ

『ディビジョン リサージェンス』のストーリーの大部分はニューヨークを中心に展開するが、その設定は**シルバークリーク**での出来事にも及んでいる。2026年2月初旬、エージェントたちは政府の重要施設である種子貯蔵施設を保護するためにこの小さな町へ派遣された。このサブプロットは、SHDの任務が単に人々を救うことだけでなく、アメリカの生物学的・農業的な未来を確保することでもあったことを強調している。

物語は、ワシントンD.C.が陥落しSHDネットワークがダウンしていく中で、シルバークリークのエージェントたちが「死守せよ」との命令を受けていたことを明らかにしている。これはニューヨークの物語と興味深い対照をなしており、ディビジョンが各セル間の通信がほとんど途絶した状態で、多方面での戦争を強いられていたことを示している。

| 場所 | 主な目的 | 2026年のステータス |
| :--- | :--- | :--- |
| **マンハッタン** | 市民・VIPの保護 | 争奪戦 / 極めて混乱 |
| **シルバークリーク** | 種子貯蔵庫の確保 | ヴァルチャーズ/リーパーズが占拠 |
| **ワシントンD.C.** | SHDネットワークの復旧 | 陥落 / ネットワークオフライン |

## 第一波（ファーストウェーブ）の役割

『ディビジョン リサージェンス』のストーリーにおける大きな魅力は、まだローグ化していない第一波エージェントとしてプレイできる点にある。オリジナルの『ディビジョン』では、第一波はしばしば裏切り（特にアーロン・キーナー）の文脈で描かれる。しかし『リサージェンス』では、ドルインフルの発生初期の数週間にわたり、自らの任務に忠実であり続けた者たちが直面した、真の英雄的行為や不可能な選択を体験することができる。

> **ヒント：** 街中に点在する「エコー（Echoes）」に注目しよう。これらは特定のJTF検問所がなぜ失われたのか、あるいはなぜ特定のVIPが狙われたのかといった、ストーリーの「欠けているピース」を埋めてくれる。

本作のシーズン制構造により、3ヶ月ごとに新しい物語コンテンツが提供される予定だ。これにより、『ディビジョン リサージェンス』のストーリーは進化し続け、SHDネットワークの停止やその後の第二波（セカンドウェーブ）の招集へと繋がる出来事まで描かれる可能性がある。

## FAQ

**Q: 『ディビジョン リサージェンス』のストーリーはタイムラインのどこに位置しますか？**

A: 主にニューヨーク市におけるグリーンポイズン発生直後の数週間を舞台にしています。第1作目の出来事と並行した物語ですが、ロウアー・マンハッタンの異なるセクターやキャラクターに焦点を当てています。

**Q: アーロン・キーナーは物語に関わっていますか？**

A: 物語はジャックスのような新しい敵対者に焦点を当てていますが、JTFが統制を失っていく中で、第一波の最終的な裏切りやキーナーのような人物の影はロア全体を通じて感じられます。

**Q: ストーリーを理解するために『ディビジョン1』や『2』をプレイしておく必要はありますか？**

A: いいえ。長年のファン向けの言及も多く含まれていますが、本作は新規プレイヤーに向けてドルインフルやSHD招集の核心的なコンセプトを解説する、独立した作品として設計されています。

**Q: ストーリーはニューヨーク以外にも広がりますか？**

A: すでにシルバークリークやワシントンD.C.での出来事にも触れられており、開発者は今後のシーズンアップデートを通じて物語の範囲をさらに拡大していく意向を示しています。
